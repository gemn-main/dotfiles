```python
class Solution(object):
    def hammingWeight(self, n):
        return str(bin(n)).count('1')
```

Для данного положительного целого числа `n`напишите функцию, которая возвращает количество

набор битов

в его двоичном представлении (также известном как [вес Хэмминга](http://en.wikipedia.org/wiki/Hamming_weight) ).

**Пример 1:**

**Вход:** n = 11

**Выход:** 3

**Объяснение:**

Входная двоичная строка **1011** имеет всего три установленных бита.

**Пример 2:**

**Вход:** n = 128

**Выход:** 1

**Объяснение:**

Входная двоичная строка **10000000** имеет всего один установленный бит.

**Пример 3:**

**Входные данные:** n = 2147483645

**Выход:** 30

**Объяснение:**

Входная двоичная строка **11111111111111111111111111111101** имеет в общей сложности тридцать установленных бит.
