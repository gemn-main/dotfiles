```python
import numpy as np
class Solution(object):
    def diagonalSum(self, mat):
        if len(mat) == 1:
            return mat[0][0]
        b = np.diag(mat)
        c = np.diag(np.fliplr(mat))
        if len(mat) % 2 == 0:
            return sum(b) + sum(c)
        else:
            return sum(b) + ((sum(c) - (c[len(c)// 2])))
```
Для данной квадратной матрицы  `mat`вернуть сумму диагоналей матрицы.

Включайте только сумму всех элементов на главной диагонали и всех элементов на второстепенной диагонали, которые не являются частью главной диагонали.

**Пример 1:**
![](https://assets.leetcode.com/uploads/2020/08/14/sample_1911.png)

**Ввод:** mat = [1 ,2, 3], 
			[4, 5,6],
			[7,8, 9]
 **Вывод:** 25
 **Пояснение:** Сумма диагоналей: 1 + 5 + 9 + 3 + 7 = 25 
Обратите внимание, что элемент mat [1, 1]= 5 учитывается только один раз.
