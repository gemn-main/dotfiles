```python
class Solution(object):
    def subarraySum(self, nums):
        o = 0
        for i in range(len(nums)):
            o += sum(nums[max(0, i - nums[i]):i+1])
        return o
```
Вам дан целочисленный массив numsразмера n. Для каждого индекса , iгде 0 <= i < n, определите
подмассив
 nums[start ... i]где start = max(0, i - nums[i]).

Возвращает общую сумму всех элементов подмассива, определенного для каждого индекса в массиве.

 

Пример 1:

Ввод: числа = [2,3,1]

Выход: 11

Объяснение:

я	Подмассив	Сумма
0	nums[0] = [2]	2
1	nums[0 ... 1] = [2, 3]	5
2	nums[1 ... 2] = [3, 1]	4
Общая сумма	 	11
Общая сумма равна 11. Следовательно, 11 — это выход.

Пример 2:

Ввод: числа = [3,1,1,2]

Выход: 13

Объяснение:

я	Подмассив	Сумма
0	nums[0] = [3]	3
1	nums[0 ... 1] = [3, 1]	4
2	nums[1 ... 2] = [1, 1]	2
3	nums[1 ... 3] = [1, 1, 2]	4
Общая сумма	 	13
Общая сумма равна 13. Следовательно, 13 — это выход.
