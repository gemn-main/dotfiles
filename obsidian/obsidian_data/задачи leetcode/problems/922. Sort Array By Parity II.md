```python
class Solution(object):
    def sortArrayByParityII(self, nums):
        a = []
        o = []
        for i in nums:
            if i % 2 == 0:
                o.append(i)
            else:
                a.append(i)
        k = []
        for i in range(len(nums) //2):
            k.append(o[i])
            k.append(a[i]) 
        return k 
```
Дан массив целых чисел `nums`, половина из которых `nums`нечетные **,** а другая половина — **четные** .

Сортируем массив так, чтобы всякий раз, когда `nums[i]`нечетное, `i`было **нечетное** , а всякий раз , когда `nums[i]`четное, `i`было **четное** .

Верните _любой массив ответов, удовлетворяющий этому условию_ .

**Пример 1:**

**Ввод:** nums = [4,2,5,7]
 **Вывод:** [4,5,2,7]
 **Пояснение:** [4,7,2,5], [2,5,4,7], [2,7,4,5] также были бы приняты.

**Пример 2:**

**Ввод:** числа = [2,3]
 **Вывод:** [2,3]
