```python
class Solution(object):
    def numberOfCuts(self, n):
        if n == 1:
            return 0
        if n % 2 == 0:
            return n / 2
        else:
            return n
```


**Допустимым разрезом** по окружности может быть:

- Разрез, представленный прямой линией, которая касается двух точек на краю круга и проходит через его центр, или
- Разрез, представленный прямой линией, которая касается одной точки на краю круга и его центра.

Некоторые допустимые и недопустимые разрезы показаны на рисунках ниже.

Дано целое число `n`, вернуть _минимальное количество разрезов **,** необходимое для разделения круга на_ `n`_равные части_ .

**Пример 1:**

![](https://assets.leetcode.com/uploads/2022/10/24/11drawio.png)

**Вход:** n = 4
 **Выход:** 2
 **Пояснение:** 
На рисунке выше показано, как, дважды разрезав круг посередине, можно получить четыре равные части.

**Пример 2:**

![](https://assets.leetcode.com/uploads/2022/10/24/22drawio.png)

**Вход:** n = 3
 **Выход:** 3
 **Пояснение:**
Чтобы разделить круг на три равные части, необходимо сделать не менее 3 разрезов.
Можно показать, что менее чем за три разреза невозможно получить три ломтика одинакового размера и формы.
Также обратите внимание, что первый разрез не разделит круг на отдельные части.
