```cpp
class Solution(object):
    def commonFactors(self, a, b):
        o = 0
        for i in range(1,min(a,b)+1):
            if a % i == 0 and b % i == 0:
                o+=1
        return o
```
Даны два положительных целых числа aи b, вернуть количество общих множителей aиb .

Целое число xявляется общим множителем и a, bесли xделит aи b.

 

Пример 1:

Вход: a = 12, b = 6
 Выход: 4
 Пояснение: Общими множителями чисел 12 и 6 являются 1, 2, 3, 6.
Пример 2:

Вход: a = 25, b = 30
 Выход: 2
 Пояснение: Общие множители чисел 25 и 30 — 1, 5.
