```python
class Solution(object):
    def sumOfMultiples(self, n):
        o = 0
        for i in range(1, n+1):
            if i % 3 == 0 or i % 5 == 0 or i % 7 == 0:
                o += i
        return o 
```

Дано положительное целое число n. Найдите сумму всех целых чисел в диапазоне [1, n] включительно , которые делятся на 3, 5или 7.

Верните целое число, обозначающее сумму всех чисел в заданном диапазоне, удовлетворяющих ограничению.

Вход: n = 7
 Выход: 21
 Пояснение: Числа в диапазоне [1, 7], которые делятся на 3, 5,или 7 являются 3, 5, 6, 7. Сумма этих чисел равна 21.
