```python
class Solution(object):
    def sortArrayByParity(self, nums):
        o = []
        a = []
        for i in nums:
            if i % 2 == 0:
                o.append(i)
            else:
                a.append(i)
        return o + a
```

Дан массив целых чисел nums. Переместите все четные целые числа в начало массива, а затем все нечетные целые числа.

Верните любой массив , удовлетворяющий этому условию .

 

Пример 1:

Вход: nums = [3,1,2,4]
 Выход: [2,4,3,1]
 Пояснение: Выходы [4,2,3,1], [2,4,1,3] и [4,2,1,3] также будут приняты.
