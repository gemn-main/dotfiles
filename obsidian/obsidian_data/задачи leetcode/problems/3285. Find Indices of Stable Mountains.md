```python
class Solution(object):
    def stableMountains(self, height, threshold):
        o = []
        for i in range(1,len(height)):
            if height[i-1] > threshold:
                o.append(i)
        return o
```

Есть nгоры в ряд, и у каждой горы есть высота. Вам дан целочисленный массив height, где height[i]представляет высоту горы i, и целое число threshold.

Гора называется устойчивой , если гора непосредственно перед ней ( если она существует ) имеет высоту строго больше . thresholdОбратите внимание , что гора 0 не является устойчивой.

Верните массив, содержащий индексы всех устойчивых гор в любом порядке.

Пример 1:

Ввод: высота = [1,2,3,4,5], порог = 2
Вывод: [3,4]

Объяснение:
Гора 3 устойчива, так как height[2] == 3больше threshold == 2.
Гора 4 стабильна, так как height[3] == 4больше threshold == 2.
