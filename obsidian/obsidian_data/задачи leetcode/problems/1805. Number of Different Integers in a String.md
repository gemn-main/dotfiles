```python
class Solution(object):
    def numDifferentIntegers(self, word):
        for i in range(len(word)):
            if word[i].isdigit():
                pass
            else:
                word = word[:i] + ' ' + word[i+1:]
        o = word.split()
        for i in range (len(o)):
            o[i] = int(o[i])
        return len(set(o))
```
Вам дана строка word, состоящая из цифр и строчных английских букв.

Вы замените каждый нецифровой символ пробелом. Например, "a123bc34d8ef34"станет " 123  34 8  34". Обратите внимание, что у вас остались некоторые целые числа, разделенные хотя бы одним пробелом: "123", "34", "8", и "34".

Верните количество различных целых чисел после выполнения операций заменыword .

Два целых числа считаются различными, если их десятичные представления без начальных нулей различны.

Вход: word = "a123bc34d8ef34"
 Выход: 3
 Пояснение: Три различных целых числа — "123", "34" и "8". Обратите внимание, что "34" учитывается только один раз.
