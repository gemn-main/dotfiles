```python
class Solution(object):
    def minimumOperations(self, nums):
        o = 0
        for i in nums:
            if i % 3 != 0:
                o += 1
        return o
```

Вам дан целочисленный массив nums. За одну операцию вы можете прибавить или вычесть 1 из любого элемента nums.

Верните минимальное количество операций, чтобы сделать все элементы numsделящимися на 3.

Пример 1:
Ввод: числа = [1,2,3,4]
Выход: 3

Объяснение:
Все элементы массива можно сделать делимыми на 3 с помощью трех операций:
Вычтите 1 из 1.
Прибавьте 1 к 2.
Вычтите 1 из 4.

Пример 2:
Ввод: числа = [3,6,9]
Выход: 0
