```cpp
class Solution(object):
    def separateDigits(self, nums):
        answer = []
        for i in range(len(nums)):
            if nums[i] >= 10:
                b =str(nums[i])
                for j in range(len(b)):
                    answer.append(int(b[j]))
            else:
                answer.append(nums[i])
        return answer
```
Дан массив положительных целых чисел `nums`, вернуть _массив_ `answer`_, состоящий из цифр каждого целого числа в_ `nums`_после разделения их в **том же порядке** , в котором они указаны в_`nums` .

Разделить цифры целого числа — значит расположить все его цифры в одном и том же порядке.

- Например, для целого числа `10921`разделение его цифр равно `[1,0,9,2,1]`.

**Пример 1:**

**Ввод:** nums = [13,25,83,77]
 **Вывод:** [1,3,2,5,8,3,7,7]
 **Пояснение:** - Разделение 13 - [1,3]. - Разделение 25 - [2,5]. - Разделение 83 - [8,3]. - Разделение 77 - [7,7]. 
ответ = [1,3,2,5,8,3,7,7]. Обратите внимание, что ответ содержит разделения в том же порядке.

**Пример 2:**

**Ввод:** nums = [7,1,3,9]
 **Вывод:** [7,1,3,9]
 **Пояснение:** Разделение каждого целого числа в nums равно самому себе. 
ответ = [7,1,3,9].
