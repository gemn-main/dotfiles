```python
class Solution(object):
    def rowAndMaximumOnes(self, mat):
        k = []
        for i in range(len(mat)):
            o = 0
            for j in range(len(mat[i])):
                if mat[i][j] == 1:
                    o += 1
            k.append([i, o])
        b = [0, 0]
        for i in range(len(k)):
            if b[1] < k[i][1]:
                b = k[i]
        return b
```

Для заданной `m x n`двоичной матрицы `mat`найдите позицию строки **с индексом 0 , которая содержит** **максимальное** количество **единиц,** и количество единиц в этой строке.

Если имеется несколько строк с максимальным количеством единиц, следует выбрать строку с **наименьшим номером .**

Возвращает _массив, содержащий индекс строки и количество единиц в ней._

**Пример 1:**

**Ввод:** mat = [0,1],[1,0]
 **Вывод:** [0,1]
 **Объяснение:** Обе строки имеют одинаковое количество единиц. Поэтому мы возвращаем индекс меньшей строки, 0, и максимальное количество единиц (1 `)`. Таким образом, ответ — [0,1]. 

**Пример 2:**

**Ввод:** mat = [0,0,0],[0,1,1]
 **Вывод:** [1,2]
 **Объяснение:** строка с индексом 1 имеет максимальное количество единиц `(2)`. Поэтому мы возвращаем ее индекс, `1`, и количество. Итак, ответ — [1,2].

**Пример 3:**

**Вход:** mat = [0,0],[1,1],[0,0]
 **Выход:** [1,2]
 **Пояснение:** строка с индексом 1 имеет максимальное количество единиц (2). Поэтому ответ — [1,2].
