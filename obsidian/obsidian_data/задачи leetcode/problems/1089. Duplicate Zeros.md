```python
class Solution(object):
    def duplicateZeros(self, arr):
        n = len(arr)
        i = 0
        while i < n:
            if arr[i] == 0:
                arr.insert(i, 0)  
                arr.pop()         
                i += 1       
            i += 1
        return arr
```
Дан массив целых чисел фиксированной длины arr. Дублируем каждое вхождение нуля, сдвигая оставшиеся элементы вправо.

Обратите внимание , что элементы, выходящие за пределы длины исходного массива, не записываются. Выполните указанные выше изменения входного массива на месте и ничего не возвращайте.

Вход: arr = [1,0,2,3,0,4,5,0]
 Выход: [1,0,0,2,3,0,0,4]
