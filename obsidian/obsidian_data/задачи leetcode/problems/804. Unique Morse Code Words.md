```python
class Solution(object):
    def uniqueMorseRepresentations(self, words):
        morse = {
            'a': '.-', 'b': '-...', 'c': '-.-.', 'd': '-..', 'e': '.',
            'f': '..-.', 'g': '--.', 'h': '....', 'i': '..', 'j': '.---',
            'k': '-.-', 'l': '.-..', 'm': '--', 'n': '-.', 'o': '---',
            'p': '.--.', 'q': '--.-', 'r': '.-.', 's': '...', 't': '-',
            'u': '..-', 'v': '...-', 'w': '.--', 'x': '-..-', 'y': '-.--',
            'z': '--..',
        }
        o = []
        for i in range(len(words)):
            k = ''
            for j in range(len(words[i])):
                k += morse[words[i][j]]
            if k not in o:
                o.append(k)
        return len(o)
```

Международная азбука Морзе определяет стандартную кодировку, в которой каждая буква сопоставляется с серией точек и тире следующим образом:

'a'карты в ".-",
'b'карты в "-...",
'c'карты в "-.-."и так далее.

Дан массив строк words, в котором каждое слово может быть записано как конкатенация кода Морзе каждой буквы.

Например, "cab"можно записать как "-.-..--...", что является конкатенацией "-.-.", ".-", и "-...". Мы будем называть такую ​​конкатенацию преобразованием слова.
