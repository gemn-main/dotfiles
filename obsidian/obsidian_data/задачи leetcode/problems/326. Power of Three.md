```python
class Solution(object):
    def isPowerOfThree(self, n):
        if n == 0:
            return False
        s = 0
        while n != 1:
            if (n // 3) * 3 == n:
                n = n // 3
                s += 1
            else:
                return False
        return True
```

Дано целое число `n`, вернуть, _`true`если оно является степенью трех. В противном случае вернуть`false`_ .

Целое число `n`является степенью числа три, если существует целое число `x`такое, что .`n == 3x`

**Пример 1:**

**Вход:** n = 27
 **Выход:** true
 **Объяснение:** 27 = 3 3

**Пример 2:**

**Вход:** n = 0
 **Выход:** false
 **Пояснение:** Нет x там, где 3 x = 0.

**Пример 3:**

**Вход:** n = -1
 **Выход:** false
 **Пояснение:** Нет x там, где 3 x = (-1).
