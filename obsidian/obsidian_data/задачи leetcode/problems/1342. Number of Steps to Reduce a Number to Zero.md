```python
class Solution(object):
    def numberOfSteps(self, num):
        o = 0
        while num > 0:
            if num % 2 == 0:
                num = num // 2
                o += 1
            else:
                num = num -1
                o += 1 
        return o
```
Дано целое число num, вернуть количество шагов, необходимых для уменьшения его до нуля .

За один шаг, если текущее число четное, его нужно разделить на 2, в противном случае нужно вычесть 1из него четное число.

 

Пример 1:

Ввод: num = 14
 Вывод: 6
 Пояснение:  
Шаг 1) 14 четное; делим на 2 и получаем 7.  
Шаг 2) 7 нечетное; вычитаем 1 и получаем 6. 
Шаг 3) 6 четное; делим на 2 и получаем 3.  
Шаг 4) 3 нечетное; вычитаем 1 и получаем 2.  
Шаг 5) 2 четное; делим на 2 и получаем 1.  
Шаг 6) 1 нечетное; вычитаем 1 и получаем 0.
