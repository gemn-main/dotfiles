```python
class Solution(object):
    def generateTheString(self, n):
        a = 'a'
        b = 'b'
        if (n % 2) == 0:
            res = a * (n-1) + b
        else:
            res = a * n
        return res
```

Для заданного целого числа `n`вернуть _строку с `n` символами, в которой каждый символ встречается **нечетное количество раз**_ .

Возвращаемая строка должна содержать только строчные английские буквы. Если допустимых строк несколько, вернуть **любую** из них.  

**Пример 1:**

**Ввод:** n = 4
 **Вывод:** "pppz"
 **Пояснение:** "pppz" — допустимая строка, поскольку символ 'p' встречается три раза, а символ 'z' встречается один раз. Обратите внимание, что существует много других допустимых строк, таких как "ohhh" и "love".

**Пример 2:**

**Ввод:** n = 2
 **Вывод:** "xy"
 **Пояснение:** "xy" — допустимая строка, поскольку символы 'x' и 'y' встречаются один раз. Обратите внимание, что существует много других допустимых строк, таких как "ag" и "ur".
