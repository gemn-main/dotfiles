```python
class Solution(object):
    def countKDifference(self, nums, k):
        o = 0
        for i in range(len(nums)):
            for _ in range(len(nums)):
                if abs(nums[i] - nums[_]) == k:
                    o += 1
        return o // 2
```

Дан массив целых чисел numsи целое число k, вернуть количество пар (i, j) , где , i < j таких что |nums[i] - nums[j]| == k .

Значение |x|определяется как:

xесли x >= 0.
-xесли x < 0.
 

Пример 1:

Ввод: nums = [1,2,2,1], k = 1
 Вывод: 4
 Пояснение: Пары с абсолютной разностью 1:
- [ 1 , 2 ,2,1]
- [ 1 ,2, 2 ,1]
- [1, 2 ,2, 1 ]
- [1,2, 2 , 1 ]
Пример 2:

Ввод: nums = [1,3], k = 3
 Вывод: 0
 Пояснение: Нет пар с абсолютной разностью 3.
Пример 3:

Ввод: nums = [3,2,1,5,4], k = 2
 Вывод: 3
 Пояснение: Пары с абсолютной разностью 2:
- [ 3 ,2, 1 ,5,4]
- [ 3 ,2,1, 5 ,4]
- [3, 2 ,1,5, 4 ]
