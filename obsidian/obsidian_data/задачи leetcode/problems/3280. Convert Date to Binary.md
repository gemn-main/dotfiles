```python
class Solution(object):
    def convertDateToBinary(self, date):
        def biin(num):
            if num[0] == '0':
                num = num[1:]
            num = int(num)
            print(num)
            return str(bin(num))[2:]
        o = biin(date[:4]) + "-" + biin(date[5:7]) + "-" + biin(date[8:])
        return o
```

Вам дана строка, date представляющая дату по григорианскому календарю в yyyy-mm-dd формате.

date можно записать в двоичном представлении, полученном путем преобразования года, месяца и дня в их двоичные представления без начальных нулей и записи их в year-month-dayформате.

Верните двоичное представление числа date.

 

Пример 1:

Ввод: дата = "2080-02-29"

Вывод: "100000100000-10-11101"

Объяснение:

100000100000, 10 и 11101 являются двоичными представлениями чисел 2080, 02 и 29 соответственно.

Пример 2:

Ввод: дата = "1900-01-01"

Вывод: "11101101100-1-1"

Объяснение:

11101101100, 1 и 1 являются двоичными представлениями чисел 1900, 1 и 1 соответственно.
