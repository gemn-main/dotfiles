```python
class Solution(object):
    def minLength(self, s):
        alphabet = {'A': 0, 'B': 1, 'C': 4, 'D': 5}
        def lolik(s):
            for e in range(len(s)-1):
                if s[e] in alphabet and s[e+1] in alphabet:
                    if alphabet[s[e]]  == (alphabet[s[e+1]])-1:
                        s = s[:e] + s[e+2:]
                        return lolik(s)
            return s
        return len(lolik(s))
```
Вам дана строка, `s`состоящая только из **заглавных** английских букв.

К этой строке можно применить некоторые операции, при помощи одной операции можно удалить **любое** вхождение одной из подстрок `"AB"`или `"CD"`из `s`.

Верните _минимально возможную длину результирующей строки, которую вы можете **получить**_ .

**Обратите внимание** , что строка объединяется после удаления подстроки и может создавать новые `"AB"`или новые `"CD"`подстроки.

**Пример 1:**

**Вход:** s = "ABFCACDB"
 **Выход:** 2
 **Пояснение:** Мы можем выполнить следующие операции: - Удалить подстроку " AB FCACDB", так что s = "FCACDB". - Удалить подстроку "FCA CD B", так что s = "FCAB". - Удалить подстроку "FC AB ", так что s = "FC". Таким образом, результирующая длина строки равна 2. 
Можно показать, что это минимальная длина, которую мы можем получить.

**Пример 2:**

**Ввод:** s = "ACBBD"
 **Вывод:** 5
 **Пояснение:** Мы не можем выполнять никаких операций со строкой, поэтому длина остается прежней.
