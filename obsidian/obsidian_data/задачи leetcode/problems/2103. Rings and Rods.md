```python
class Solution(object):
    def countPoints(self, rings):
        def one_rings(num, rings):
            o = ""
            for i in range(len(rings)):
                if rings[i] == num:
                    if rings[i-1] in o:
                        pass
                    else:
                        o += rings[i-1]
            return len(o)
        b = 0
        for i in range(10):
            if 3 <= one_rings(str(i), rings):
                b += 1
        return b
```

Есть `n`кольца, и каждое кольцо либо красное, либо зеленое, либо синее. Кольца распределены **по десяти стержням,** помеченным от `0`до `9`.

Вам дана строка `rings`длины `2n`, описывающая `n`кольца, которые размещены на стержнях. Каждые два символа в `rings`образуют **пару цвет-позиция** , которая используется для описания каждого кольца, где:

- Первый символ пары обозначает **цвет** кольца ( , , ) **.**`ith``ith``'R'``'G'``'B'`
- Второй символ пары обозначает **стержень** , на который надевается кольцо ( к ) **.**`ith``ith``'0'``'9'`

Например, `"R3G2B1"`описывает `n == 3`кольца: красное кольцо, надетое на стержень с меткой 3, зеленое кольцо, надетое на стержень с меткой 2, и синее кольцо, надетое на стержень с меткой 1.

Верните _количество стержней, на которых есть кольца **всех трех цветов .**_

**Пример 1:**

![](https://assets.leetcode.com/uploads/2021/11/23/ex1final.png)

**Вход:** rings = "B0B6G0R6R0R6G9"
 **Выход:** 1
 **Объяснение:** - Стержень с меткой 0 содержит 3 кольца всех цветов: красное, зеленое и синее. - Стержень с меткой 6 содержит 3 кольца, но на нем только красное и синее. - Стержень с меткой 9 содержит только зеленое кольцо. 
Таким образом, количество стержней всех трех цветов равно 1.

**Пример 2:**

![](https://assets.leetcode.com/uploads/2021/11/23/ex2final.png)

**Вход:** rings = "B0R0G0R9R0B0G0"
 **Выход:** 1
 **Пояснение:** - Стержень с меткой 0 содержит 6 колец всех цветов: красного, зеленого и синего. - Стержень с меткой 9 содержит только красное кольцо. 
Таким образом, количество стержней всех трех цветов равно 1.

**Пример 3:**

**Вход:** кольца = "G4"
 **Выход:** 0
 **Пояснение:**  
Дано только одно кольцо. Таким образом, ни один стержень не имеет всех трех цветов.
