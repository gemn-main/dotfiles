```python
class Solution(object):
    def canAliceWin(self, n):
        k = 11
        while True:
            k = k - 1   
            if k % 2 == 0:
                n -= k 
                if n < 0:
                    return False
            else:
                n -= k 
                if n < 0:
                    return True 
```

Алиса и Боб играют в игру, в которой они по очереди вынимают камни из кучи, причем Алиса ходит первой .

Алиса начинает с того, что за свой первый ход убирает ровно 10 камней.
За каждый последующий ход каждый игрок убирает ровно на 1 камень меньше, чем предыдущий противник.
Игрок, который не может сделать ход, проигрывает.

Дано положительное целое число n, вернуть, trueесли Алиса выиграет игру, и falseв противном случае.

 

Пример 1:

Вход: n = 12

Вывод: истина

Объяснение:

Алиса убирает 10 камней в свой первый ход, оставляя 2 камня Бобу.
Боб не может убрать 9 камней, поэтому Алиса выигрывает.
Пример 2:

Вход: n = 1

Вывод: ложь

Объяснение:

Алиса не может убрать 10 камней, поэтому Алиса проигрывает.
