```python
class Solution(object):
    def countLargestGroup(self, n):
        def sumn(num):
            num = str(num)
            o = 0
            for i in range(len(num)):
                o += int(num[i])
            return o
        a = {}
        for i in range(1,n+1):
            k = sumn(i) 
            if k in a:
                a[k] += 1
            else:
                a[k] = 1
        o = [0,0]
        for i in a.values():
            if i > o[0]:
                o = [i, 1]
            else:
                if i == o[0]:
                    o[1] += 1
        return o[1]
```
Вам дано целое число n.

Каждое число от 1до nгруппируется по сумме его цифр.

Верните количество групп, имеющих наибольший размер .

 

Пример 1:

Вход: n = 13
 Выход: 4
 Пояснение: Всего имеется 9 групп, они сгруппированы по сумме цифр чисел от 1 до 13: 
[1,10], [2,11], [3,12], [4,13], [5], [6], [7], [8], [9]. 
Имеется 4 группы с наибольшим размером.
Пример 2:

Вход: n = 2
 Выход: 2
 Пояснение: Имеется 2 группы [1], [2] размера 1.
