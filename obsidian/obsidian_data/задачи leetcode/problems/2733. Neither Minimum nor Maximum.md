```python
class Solution(object):
    def findNonMinOrMax(self, nums):
        nums = list(set(nums))
        if len(nums) <=2:
            return -1
        else:
            nums.sort()
            return nums[1]
```
Дан массив целых чисел , numsсодержащий различные положительные целые числа. Найдите и верните любое число из массива, которое не является ни минимальным , ни максимальным значением в массиве, или -1если такого числа нет.

Верните выбранное целое число.

 

Пример 1:

Ввод: nums = [3,2,1,4]
 Вывод: 2
 Пояснение: В этом примере минимальное значение равно 1, а максимальное значение равно 4. Следовательно, допустимыми ответами могут быть 2 или 3.
Пример 2:

Ввод: nums = [1,2]
 Вывод: -1
 Пояснение: Поскольку в nums нет числа, которое не является ни максимальным, ни минимальным, мы не можем выбрать число, удовлетворяющее данному условию. Следовательно, ответа нет.
Пример 3:

Ввод: nums = [2,1,3]
 Вывод: 2
 Пояснение: Поскольку 2 не является ни максимальным, ни минимальным значением в nums, это единственный допустимый ответ.
