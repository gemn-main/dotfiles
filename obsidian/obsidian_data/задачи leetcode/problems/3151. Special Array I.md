```python
class Solution(object):
    def isArraySpecial(self, nums):
        if len(nums) == 1:
            return True
        for i in range(len(nums)-1):
            if nums[i] % 2 == nums[i+1] % 2:
                return False
        return True
```
Массив считается **особым** , если каждая пара его соседних элементов содержит два числа с разной четностью.

Вам дан массив целых чисел `nums`. Верните, `true`если `nums`это **специальный** массив, в противном случае верните `false`.

**Пример 1:**

**Ввод:** числа = [1]

**Вывод:** истина

**Объяснение:**

Есть только один элемент. Поэтому ответ `true`.

**Пример 2:**

**Ввод:** числа = [2,1,4]

**Вывод:** истина

**Объяснение:**

Есть только две пары: `(2,1)`и `(1,4)`, и обе они содержат числа с разной четностью. Поэтому ответ `true`.

**Пример 3:**

**Ввод:** числа = [4,3,1,6]

**Вывод:** ложь

**Объяснение:**

`nums[1]`и `nums[2]`оба нечетные. Так что ответ `false`.
