```python
class Solution(object):
    def findTheDistanceValue(self, arr1, arr2, d):
        o = 0
        for i in range(len(arr1)):
            a = 0
            for j in range(len(arr2)):
                if abs(arr1[i] - arr2[j]) <= d:
                    a = -1

            if a == 0:
                o += 1
        return o
```

Даны два целочисленных массива arr1и arr2, а также целое число d, вернуть значение расстояния между двумя массивами .

Значение расстояния определяется как количество элементов arr1[i], при котором нет ни одного элемента, arr2[j]где |arr1[i]-arr2[j]| <= d.

 

Пример 1:

Вход: arr1 = [4,5,8], arr2 = [10,9,1,8], d = 2
 Выход: 2
 Объяснение:  
Для arr1[0]=4 имеем: 
|4-10|=6 > d=2 
|4-9|=5 > d=2 
|4-1|=3 > d=2 
|4-8|=4 > d=2 
Для arr1[1]=5 имеем: 
|5-10|=5 > d=2 
|5-9|=4 > d=2 
|5-1|=4 > d=2 
|5-8|=3 > d=2 
Для arr1[2]=8 имеем: |8-10|=2 <= d=2 |8-9|=1 <= d=2 
|8-1|=7 > d=2 |8-8|=0 <= d=2

 Выход: 2
