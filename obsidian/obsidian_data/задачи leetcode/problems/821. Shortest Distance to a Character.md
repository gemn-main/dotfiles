```python
class Solution(object):
    def shortestToChar(self, s, c):
        arr = [float('inf')] * len(s)
        for i in range(len(s)):
            if s[i] == c:
                for j in range(len(s)):
                    o = abs(i-j)
                    arr[j] = min(o, arr[j])
        return arr
```

Для заданной строки `s`и символа `c`, встречающегося в `s`, вернуть _массив целых чисел,_ `answer`_где_ `answer.length == s.length`_и_ `answer[i]`_— **расстояние** от индекса_ `i`_до **ближайшего** вхождения символа_ `c`_в_`s` .

Расстояние между двумя индексами и равно **,** где — функция абсолютного значения.`i``j``abs(i - j)``abs`

**Пример 1:**

**Ввод:** s = "loveleetcode", c = "e"
 **Вывод:** [3,2,1,0,1,0,0,1,2,2,1,0]
 **Пояснение:** Символ "e" появляется в индексах 3, 5, 6 и 11 (индексация начинается с 0).
Ближайшее вхождение «e» для индекса 0 находится в индексе 3, поэтому расстояние равно abs(0 - 3) = 3.
Ближайшее вхождение «e» для индекса 1 находится в индексе 3, поэтому расстояние равно abs(1 - 3) = 2.
Для индекса 4 существует связь между «e» в индексе 3 и «e» в индексе 5, но расстояние все равно остается прежним: abs(4 - 3) == abs(4 - 5) = 1.
Ближайшее вхождение «e» для индекса 8 находится под индексом 6, поэтому расстояние равно abs(8 - 6) = 2.
