```python
class Solution(object):
    def kidsWithCandies(self, candies, extraCandies):
        max_cand = max(candies)
        o = []
        for i in candies:
            if i + extraCandies >= max_cand:
                o.append(True)
            else:
                o.append(False)
        return o
```
Есть nдети с конфетами. Вам дан массив целых чисел candies, где каждое число candies[i]представляет количество конфет у ребенка, и целое число , обозначающее количество дополнительных конфет, которые есть у вас.ithextraCandies

Возвращает логический массив resultдлины n, где result[i]— trueесли после раздачи ребенку всех , у него будет наибольшее количество конфет среди всех детей , или в противном случае .ithextraCandiesfalse

Обратите внимание, что наибольшее количество конфет может достаться нескольким детям .

 

Пример 1:

Вход: конфеты = [2,3,5,1,3], extraCandies = 3
 Выход: [true,true,true,false,true] 
 Объяснение: Если вы дадите все extraCandies: 
- Ребёнку 1, у него будет 2 + 3 = 5 конфет, что является наибольшим количеством среди детей. 
- Ребёнку 2, у него будет 3 + 3 = 6 конфет, что является наибольшим количеством среди детей. 
- Ребёнку 3, у него будет 5 + 3 = 8 конфет, что является наибольшим количеством среди детей. 
- Ребёнку 4, у него будет 1 + 3 = 4 конфеты, что не является наибольшим количеством среди детей. 
- Ребёнку 5, у него будет 3 + 3 = 6 конфет, что является наибольшим количеством среди детей.
Пример 2:

Вход: конфеты = [4,2,1,1,2], extraCandies = 1
 Выход: [true,false,false,false,false] 
 Объяснение: Есть только 1 дополнительная конфета. 
Ребенок 1 всегда будет иметь наибольшее количество конфет, даже если другой ребенок получит дополнительную конфету.
