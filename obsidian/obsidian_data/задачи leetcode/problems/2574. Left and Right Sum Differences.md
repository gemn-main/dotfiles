```cpp
class Solution {
public:
    vector<int> leftRightDifference(vector<int>& nums) {
        int total_sum =  accumulate(nums.begin(), nums.end(), 0);
        int left_sum = 0;
        vector<int> answer;
        for(int i =0; i <= nums.size()-1; i++){
            answer.push_back(abs( (total_sum - left_sum - nums[i]) - left_sum) );
            left_sum += nums[i];
        }
        return answer;
    }
};
```

Вам дан целочисленный массив с индексом 0nums размером n.

Определим два массива leftSumи rightSumгде:

leftSum[i]сумма элементов слева от индекса iв массиве nums. Если такого элемента нет, leftSum[i] = 0.
rightSum[i]сумма элементов справа от индекса iв массиве nums. Если такого элемента нет, rightSum[i] = 0.
Возвращает целочисленный массив answerразмером n.answer[i] = |leftSum[i] - rightSum[i]|

 

Пример 1:

Вход: nums = [10,4,8,3]
 Выход: [15,1,11,22]
 Объяснение: Массив leftSum равен [0,10,14,22], а массив rightSum равен [15,11,3,0]. 
Ответ массива равен [|0 - 15|,|10 - 11|,|14 - 3|,|22 - 0|] = [15,1,11,22].
Пример 2:

Вход: nums = [1]
 Выход: [0]
 Объяснение: Массив leftSum равен [0], а массив rightSum равен [0]. 
Ответ массива равен [|0 - 0|] = [0].
