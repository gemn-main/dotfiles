```python
class Solution(object):
    def minMovesToSeat(self, seats, students):
        seats.sort()
        students.sort()
        o = 0
        for i in range(len(seats)):
            o += abs(seats[i] - students[i])
        return o
```

Обратите внимание, что вначале на одной и той же позиции может находиться несколько мест или учеников .

 

Пример 1:

Вход: места = [3,1,5], студенты = [2,7,4]
 Выход: 4
 Пояснение: Студенты перемещаются следующим образом: 
- Первый студент перемещается из позиции 2 в позицию 1 с помощью 1 хода. 
- Второй студент перемещается из позиции 7 в позицию 5 с помощью 2 ходов. 
- Третий студент перемещается из позиции 4 в позицию 3 с помощью 1 хода. 
Всего было использовано 1 + 2 + 1 = 4 хода.
Пример 2:

 Пояснение: Студенты перемещаются следующим образом: 
- Первый студент не перемещается. 
- Второй студент перемещается с позиции 3 на позицию 4 с помощью 1 хода.

- Четвертый студент перемещается с позиции 6 на позицию 9 с помощью 3 ходов. 
Всего было использовано 0 + 1 + 3 + 3 = 7 ходов.
