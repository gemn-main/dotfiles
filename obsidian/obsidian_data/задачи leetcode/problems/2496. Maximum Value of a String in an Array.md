```python
class Solution(object):
    def maximumValue(self, strs):
        k = 0
        for i in range(len(strs)):
            if strs[i].isdigit():
                if k < int(strs[i]):
                    k = int(strs[i])
            else:
                if k < len(strs[i]):
                    k = len(strs[i])
        return k
```
Значение **буквенно** -цифровой строки можно определить как:

- Числовое представление строки в системе счисления base , если она состоит **только** из **цифр** .`10`
- **В противном случае — длина** строки .

Для заданного массива `strs`буквенно-цифровых строк вернуть _максимальное **значение** любой строки в_`strs` .

**Пример 1:**

**Вход:** strs = ["alic3","bob","3","4","00000"]
 **Выход:** 5
 **Пояснение:** - "alic3" состоит как из букв, так и из цифр, поэтому его значение равно его длине, т. е. 5. - "bob" состоит только из букв, поэтому его значение также равно его длине, т. е. 3. - "3" состоит только из цифр, поэтому его значение равно его числовому эквиваленту, т. е. 3. - "4" также состоит только из цифр, поэтому его значение равно 4. - "00000" состоит только из цифр, поэтому его значение равно 0. 
Следовательно, максимальное значение равно 5 для "alic3".

**Пример 2:**

**Ввод:** strs = ["1","01","001","0001"]
 **Вывод:** 1
 **Пояснение:**  
Каждая строка в массиве имеет значение 1. Следовательно, мы возвращаем 1.
