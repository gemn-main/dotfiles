```python
class Solution(object):
    def maximumCount(self, nums):
        a = 0
        o = 0
        for i in nums:
            if i > 0:
                a +=1 
            if i < 0:
                o += 1
        return max(a,o)
```
Дан массив, numsотсортированный в неубывающем порядке, вернуть максимальное значение между числом положительных целых чисел и числом отрицательных целых чисел.

Другими словами, если количество положительных целых чисел numsравно pos, а количество отрицательных целых чисел равно neg, то вернуть максимальное из posи neg.
Обратите внимание , что это 0не является ни положительным, ни отрицательным.

 

Пример 1:

Ввод: nums = [-2,-1,-1,1,2,3]
 Вывод: 3
 Пояснение: Есть 3 положительных целых числа и 3 отрицательных целых числа. Максимальное количество среди них — 3.
Пример 2:

Ввод: nums = [-3,-2,-1,0,0,1,2]
 Вывод: 3
 Пояснение: Есть 2 положительных целых числа и 3 отрицательных целых числа. Максимальное количество среди них — 3.
Пример 3:

Ввод: nums = [5,20,66,1314]
 Вывод: 4
 Пояснение: Есть 4 положительных целых числа и 0 отрицательных целых чисел. Максимальное количество среди них — 4.
