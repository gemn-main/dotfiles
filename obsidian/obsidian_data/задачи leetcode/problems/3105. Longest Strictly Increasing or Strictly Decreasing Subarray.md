class Solution(object):
    def longestMonotonicSubarray(self, nums):
        len_nums = len(nums)
        o = 1
        k = 1
        for i in range(len_nums):
            if i != len_nums-1:
                if nums[i] > nums[i+1]:
                    k += 1
                else:
                    k = 1
                if o < k:
                    o = k
        k = 1
        for i in range(len_nums):
            if i != len_nums-1:
                if nums[i] < nums[i+1]:
                    k += 1
                else:
                    k = 1
                if o < k:
                    o = k
        return o

Вам дан массив целых чисел nums. Верните длину самого длинного 
подмассив
из numsкоторых либо
строго увеличивается
или
строго убывающий
.

 

Пример 1:

Ввод: числа = [1,4,3,3,2]

Выход: 2

Объяснение:

Строго возрастающие подмассивы — numsэто [1], [2], [3], [3], [4]и [1,4].

Строго убывающие подмассивы — numsэто [1], [2], [3], [3], [4], [3,2]и [4,3].

Поэтому мы возвращаемся 2.

Строго убывающие подмассивы — numsэто [3], [3], [3], и [3].

Поэтому мы возвращаемся 1.
