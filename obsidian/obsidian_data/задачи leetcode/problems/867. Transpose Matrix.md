```cpp
class Solution {
public:
    vector<vector<int>> transpose(vector<vector<int>>& matrix) {
        int m = matrix.size();    // количество строк
        int n = matrix[0].size(); // количество столбцов
        vector<vector<int>> matrix2(n, vector<int>(m, 0));
        for (int i = 0; i < m; i++){
            for (int j = 0; j < n; j++){
                matrix2[j][i] = matrix[i][j];
            
            }
        }
        return matrix2;

    }
};
```



Дан двумерный целочисленный массив , `matrix`вернуть _транспонированный **массив** ._ `matrix`

Транспонирование матрицы — это переворачивание матрицы относительно ее главной диагонали, при **котором** индексы строк и столбцов матрицы меняются местами.

**Пример 1:**

**Вход:** матрица = [1,2,3],[4,5,6],[7,8,9]
 **Выход:** [1,4,7],[2,5,8],[3,6,9]

**Пример 2:**

**Вход:** матрица = [1,2,3],[4,5,6]
 **Выход:** [1,4],[2,5],[3,6]
