```python
class Solution(object):
    def maxSubsequence(self, nums, k):
        num = nums[:]
        o = []
        ind = []
        a = []
        nuum = nums[:]
        for i in range(k):
            o.append(max(num))
            num.pop(num.index(max(num)))

        for i in range(len(o)):
            ind.append(nuum.index(o[i]))
            nuum[nuum.index(o[i])] = "шерсть"

        for i in range(len(ind)):
            a.append(nums[min(ind)])
            ind.remove(min(ind))

        return a
```
Вам дан массив целых чисел `nums`и целое число `k`. Вы хотите найти **подпоследовательность** длины `nums`, `k`которая имеет **наибольшую** сумму.

Возвращает _**любую**_ _такую ​​подпоследовательность как целочисленный массив длины_ . `k`

Подпоследовательность — **это** массив, который может быть получен из другого массива путем удаления некоторых или ни одного элемента без изменения порядка оставшихся элементов.

**Пример 1:**

**Вход:** nums = [2,1,3,3], k = 2
 **Выход:** [3,3]
 **Пояснение:** 
Подпоследовательность имеет наибольшую сумму 3 + 3 = 6.

**Пример 2:**

**Вход:** nums = [-1,-2,3,4], k = 3
 **Выход:** [-1,3,4]
 **Пояснение:**  
Подпоследовательность имеет наибольшую сумму -1 + 3 + 4 = 6.

**Пример 3:**

**Вход:** nums = [3,4,3,3], k = 2
 **Выход:** [3,4]
 **Пояснение:** 
Подпоследовательность имеет наибольшую сумму 3 + 4 = 7. 
Другая возможная подпоследовательность — [4, 3].
