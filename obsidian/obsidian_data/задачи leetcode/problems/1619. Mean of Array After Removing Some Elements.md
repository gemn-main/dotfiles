```python
import numpy
class Solution(object):
    def trimMean(self, arr):
        five_percent = len(arr) // 20
        arr.sort()
        return numpy.mean(arr[five_percent:-five_percent])
```
Для заданного целочисленного массива arrвернуть среднее значение оставшихся целых чисел после удаления наименьшего 5%и наибольшего 5%элементов.

Ответы, не выходящие за рамки фактического ответа, будут считаться принятыми.10-5

 

Пример 1:

Вход: arr = [1,2,2,2,2,2,2,2,2,2,2,2,2,2,2,2,2,2,2,2,3]
 Выход: 2.00000
 Пояснение: После удаления минимального и максимального значений этого массива все элементы равны 2, поэтому среднее значение равно 2.
