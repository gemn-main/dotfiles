```python
class Solution(object):
    def isLongPressedName(self, name, typed):
        def ar(arr):
            a = [["lol"]]
            for i in range(len(arr)):
                if arr[i] not in a[-1]:
                    a.append([arr[i]])
                else:
                    a[-1].append(arr[i])
            return a[1:]
        name = ar(name)
        typed = ar(typed)
        if len(name) != len(typed):
            return False
        for i in range(len(name)):
            if name[i][0] != typed[i][0]:
                return False
            if len(name[i]) > len(typed[i]):
                return False
        return True
```

Вы изучаете typedсимволы клавиатуры. Вернитесь, Trueесли возможно, что это было имя вашего друга, с некоторыми символами (возможно, ни одного) при длительном нажатии.

 

Пример 1:

Ввод: имя = "alex", введено = "aaleex"
 Вывод: true
 Пояснение: «a» и «e» в «alex» были нажаты долго.
Пример 2:

Ввод: имя = "saeed", введено = "ssaaedd"
 Вывод: false
 Пояснение: должно быть, «e» было нажато дважды, но в введенном выводе этого не было.
