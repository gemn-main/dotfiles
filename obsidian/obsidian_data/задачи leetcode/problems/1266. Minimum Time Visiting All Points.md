```python
class Solution(object):
    def minTimeToVisitAllPoints(self, points):
        o = 0
        for i in range(len(points)-1):
            o += max(abs(points[i+1][0] - points[i][0]),
		              abs(points[i+1][1] - points[i][1]))
        return o
```

На 2D-плоскости есть `n`точки с целочисленными координатами . Верните _минимальное_ _**время**_ _в секундах для посещения всех точек в порядке, заданном_ .`points[i] = [xi, yi]``points`

Двигаться можно согласно следующим правилам:

- Во `1`-вторых, вы можете:
    - переместиться вертикально на одну единицу,
    - переместиться горизонтально на одну единицу, или
    - переместите `sqrt(2)`юниты по диагонали (другими словами, переместите один юнит по вертикали, а затем один по горизонтали за `1`секунду).
- Вам необходимо посетить точки в том же порядке, в котором они указаны в массиве.
- Вам разрешено проходить через пункты, которые указаны позже в заказе, но они не считаются посещениями.

**Пример 1:**

![](https://assets.leetcode.com/uploads/2019/11/14/1626_example_1.PNG)

**Вход:** точки = [1,1],[3,4],[-1,0]
 **Выход:** 7
 **Пояснение:** Один оптимальный путь — **[1,1]** -> [2,2] -> [3,3] -> **[3,4]** -> [2,3] -> [1,2] -> [0,1] -> **[-1,0]**   
Время от [1,1] до [3,4] = 3 секунды
Время от [3,4] до [-1,0] = 4 секунды
Общее время = 7 секунд
