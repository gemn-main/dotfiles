```python
class Solution(object):
    def arraySign(self, nums):
        def signFunc(x):
            if x > 0:
                return 1
            elif x < 0:
                return -1
            return 0
        o = 1
        for i in nums:
            o = o * i 
    
        return signFunc(o)
```
Реализуйте функцию signFunc(x), которая возвращает:

1если xположительно.
-1если xотрицательно.
0если xравно 0.
Вам дан массив целых чисел nums. Пусть productбудет произведением всех значений в массиве nums.

Возвращаться signFunc(product).

 

Пример 1:

Ввод: nums = [-1,-2,-3,-4,3,2,1]
 Вывод: 1
 Пояснение: Произведение всех значений в массиве равно 144, а signFunc(144) = 1
Пример 2:

Ввод: nums = [1,5,0,2,-3]
 Вывод: 0
 Пояснение: Произведение всех значений в массиве равно 0, а signFunc(0) = 0
Пример 3:

Ввод: nums = [-1,1,-1,1,-1]
 Вывод: -1
 Пояснение: Произведение всех значений в массиве равно -1, а signFunc(-1) = -1
