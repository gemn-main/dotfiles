```python
class Solution(object):
    def generateKey(self, num1, num2, num3):
        num1 = str(num1)
        num2 = str(num2)
        num3 = str(num3)
        def transformation(num):
            len_num = len(num)
            if len_num < 4:
                need = 4 - len_num 
                num = "0" * need + num
                return num
            else:
                return num
            
        num1 = transformation(num1)
        num2 = transformation(num2)
        num3 = transformation(num3)
        k = ''
        for i in range(4):
            k += str(min(int(num1[i]), int(num2[i]),int(num3[i])))
        return int(k)
```

Вам даны три положительных целых числа num1, num2, и num3.

Число , , и определяется как четырехзначное число, такое что key:num1num2num3

Первоначально, если какое-либо число имеет менее четырех цифр, оно дополняется ведущими нулями .
Цифра ( ) числа получается путем взятия наименьшей цифры из цифр чисел , , и .ith1 <= i <= 4keyithnum1num2num3
