```python
class Solution(object):
    def finalValueAfterOperations(self, operations):
        o = 0 
        for i in range(len(operations)):
            if "+" in operations[i]:
                o += 1
            else:
                o -= 1
        return o 
```

Существует язык программирования, содержащий всего четыре операции и одну переменную X:

++Xи X++ увеличивает значение переменной Xна 1.
--Xи X-- уменьшает значение переменной Xна 1.
Первоначально значение Xравно 0.

Дан массив строк ,operations содержащий список операций, вернуть конечное значение X после выполнения всех операций .

 

Пример 1:

Ввод: операции = ["--X","X++","X++"]
 Вывод: 1
 Пояснение:  Операции выполняются следующим образом:
Первоначально X = 0.
--X: X уменьшается на 1, X = 0 - 1 = -1.
X++: X увеличивается на 1, X = -1 + 1 = 0.
X++: X увеличивается на 1, X = 0 + 1 = 1.
Пример 2:

Ввод: операции = ["++X","++X","X++"]
 Вывод: 3
 Пояснение: Операции выполняются следующим образом:
Первоначально X = 0.
++X: X увеличивается на 1, X = 0 + 1 = 1.
++X: X увеличивается на 1, X = 1 + 1 = 2.
X++: X увеличивается на 1, X = 2 + 1 = 3.
Пример 3:

Ввод: операции = ["X++","++X","--X","X--"]
 Вывод: 0
 Пояснение:  Операции выполняются следующим образом:
Первоначально X = 0.
X++: X увеличивается на 1, X = 0 + 1 = 1.
++X: X увеличивается на 1, X = 1 + 1 = 2.
--X: X уменьшается на 1, X = 2 - 1 = 1.
X--: X уменьшается на 1, X = 1 - 1 = 0.
