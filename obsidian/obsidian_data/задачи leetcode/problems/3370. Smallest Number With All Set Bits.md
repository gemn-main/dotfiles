```python
class Solution(object):
    def smallestNumber(self, n):
        o = "1"
        while True:
            new_o = int(o, 2)
            if new_o >= n:
                return new_o
            else:
                o += "1"
```
Вам дано _положительное_ число `n`.

Возвращает **наименьшее** число, `x` **большее** или **равное** `n` , такое, что двоичное представление `x`содержит только

набор битов

**Пример 1:**

**Вход:** n = 5

**Выход:** 7

**Объяснение:**

Двоичное представление числа 7 — `"111"`.

**Пример 2:**

**Вход:** n = 10

**Выход:** 15

**Объяснение:**

Двоичное представление числа 15 — `"1111"`.

**Пример 3:**

**Вход:** n = 3

**Выход:** 3

**Объяснение:**

Двоичное представление числа 3 — `"11"`.
