```python
class Solution(object):
    def maximum69Number (self, num):
        b = str(num)
        if '6' in b:
            for i in range(len(b)):
                if b[i] == '6':
                    b = b[:i] + '9' + b[(i+1):]
                    return int(b)
        else:
            return num
```
Вам дано положительное целое число, numсостоящее только из цифр 6и 9.

Верните максимальное число, которое вы можете получить, изменив не более одной цифры ( 6становится 9, а 9становится 6) .

 

Пример 1:

Ввод: num = 9669
 Вывод: 9969
 Пояснение:  
Изменение первой цифры дает результат 6669. 
Изменение второй цифры дает результат 9969. 
Изменение третьей цифры дает результат 9699. 
Изменение четвертой цифры дает результат 9666. 
Максимальное число — 9969.
Пример 2:

Ввод: num = 9996
 Вывод: 9999
 Пояснение: Изменение последней цифры 6 на 9 дает максимальное число.
Пример 3:

Ввод: num = 9999
 Вывод: 9999
 Пояснение: Лучше не применять никаких изменений.
