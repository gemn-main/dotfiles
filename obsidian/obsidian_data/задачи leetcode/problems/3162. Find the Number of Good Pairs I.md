```python
class Solution(object):
    def numberOfPairs(self, nums1, nums2, k):
        o = []
        for i in range(len(nums1)):
            for _ in range(len(nums2)):
                if nums1[i] % (nums2[_] * k) == 0:
                    o.append([i, _])
        return len(o)
```


Вам даны 2 целочисленных массива `nums1`и `nums2`длины `n`и `m`соответственно. Вам также дано **положительное** целое число `k`.

Пара `(i, j)`называется **хорошей** , если `nums1[i]`делится на `nums2[j] * k`( `0 <= i <= n - 1`, `0 <= j <= m - 1`).

Верните общее количество **хороших** пар


Пример 1:

Ввод: nums1 = [1,3,4], nums2 = [1,3,4], k = 1
Выход: 5
Объяснение:
Пять хороших пар — это (0, 0), (1, 0), (1, 1), (2, 0), и (2, 2).
Пример 2:

Ввод: nums1 = [1,2,4,12], nums2 = [2,4], k = 3
Выход: 2
Объяснение:
Две хорошие пары — это (3, 0)и (3, 1).
