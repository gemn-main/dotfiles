```python
class Solution(object):
    def balancedStringSplit(self, s):
        a = 0
        o = 0
        for i in range(len(s)):
            a += 1 if s[i] == 'R' else - 1
            if a == 0:
                o += 1
        return o
```
**Сбалансированными** называются строки, содержащие одинаковое количество символов `'L'`и `'R'`.

Дана **сбалансированная** строка `s`, разбейте ее на некоторое количество подстрок так, чтобы:

- Каждая подстрока сбалансирована.

Верните _максимальное количество сбалансированных строк, которое вы можете получить **.**_

**Пример 1:**

**Ввод:** s = "RLRRLLRLRL"
 **Вывод:** 4
 **Пояснение:** s можно разбить на "RL", "RRLL", "RL", "RL", каждая подстрока содержит одинаковое количество символов "L" и "R".

**Пример 2:**

**Ввод:** s = "RLRRRLLRLL"
 **Вывод:** 2
 **Пояснение:** s можно разбить на "RL", "RRRLLRLL", каждая подстрока содержит одинаковое количество символов "L" и "R".
Обратите внимание, что s нельзя разбить на «RL», «RR», «RL», «LR», «LL», поскольку 2 -я и 5 -я подстроки не сбалансированы.

**Пример 3:**

**Ввод:** s = "LLLLRRRR"
 **Вывод:** 1
 **Пояснение:** s можно разделить на "LLLLRRRR".
