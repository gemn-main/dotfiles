```python
class Solution(object):
    def convertToBase7(self, num):
        k = False
        if num < 0:
            k = True
            num = abs(num)
        o = ''
        while True:
            if num < 7:
                o += str(num % 7)
                if k:
                    return "-" + o[::-1]   
                return o[::-1]
            o += str(num % 7)
            num = num // 7
```
Для заданного целого числа numвернуть строку его представления в системе счисления с основанием 7 .

 

Пример 1:

Ввод: число = 100
 Вывод: "202"
Пример 2:

Вход: число = -7
 Выход: "-10"
