```python
class Solution(object):
    def areOccurrencesEqual(self, s):
        d = {}
        for i in s:
            d[str(i)] = 0
        for i in s:
            d[str(i)] += 1
        for i in d:
            if d[s[0]] != d[i]: 
                return False
            
        return True
```
Если задана строка s, вернуть , trueесли sэто хорошая строка, falseв противном случае .

Строка sсчитается хорошей , если все символы, которые в ней встречаются, sвстречаются одинаковое количество раз (т. е. с одинаковой частотой).

 

Пример 1:

Ввод: s = "abacbc"
 Вывод: true
 Пояснение: Символы, которые появляются в s, это 'a', 'b' и 'c'. Все символы встречаются в s 2 раза.
Пример 2:

Ввод: s = "aaabb"
 Вывод: false
 Пояснение: Символы, которые появляются в s, — это «a» и «b».
«a» встречается 3 раза, а «b» — 2 раза, что не равно количеству раз.
