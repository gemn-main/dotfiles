```python
class Solution(object):
    def intersection(self, nums):
        o = 0
        for i in range(len(nums)):
            if len(nums[i]) > o:
                o = i
        b = []
        for i in range(len(nums[o])):
            k = nums[o][i]
            flack = True
            for j in range(len(nums)):
                if k not in nums[j]:
                    flack = False
                    break
            if flack:
                b.append(k)
        b.sort()     
        return b
```
Дан двумерный целочисленный массив nums, где nums[i]— непустой массив различных положительных целых чисел. Вернуть список целых чисел, присутствующих в каждом массиве , numsотсортированный по возрастанию .
 

Пример 1:

Ввод: nums = [ 3 ,1,2, 4 ,5],[1,2, 3 , 4 ],[ 3 , 4 ,5,6]
 Вывод: [3,4]
 Пояснение:  
Единственные целые числа, присутствующие в каждом из nums[0] = [ 3 ,1,2, 4 ,5], nums[1] = [1,2, 3 , 4 ] и nums[2] = [ 3 , 4 ,5,6], — это 3 и 4, поэтому мы возвращаем [3,4].
Пример 2:

Ввод: nums = [1,2,3],[4,5,6]
 Вывод: []
 Пояснение:  
Не существует ни одного целого числа, присутствующего как в nums[0], так и в nums[1], поэтому мы возвращаем пустой список [].
