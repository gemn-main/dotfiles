```python
class Solution(object):
    def prefixCount(self, words, pref):
        o = 0
        pref_len = len(pref)
        for i in range(len(words)):
            if words[i][:pref_len] == pref:
                o += 1
        return o
```

Вам дан массив строк `words`и строка `pref`.

Возвращает _количество строк,_ `words`_содержащих_ `pref`_в качестве **префикса**_ .

Префиксом строки **является**`s` любая начальная непрерывная подстрока `s`.

**Пример 1:**

**Ввод:** слова = ["pay"," **at** tention","practice"," **at** tendency"], `pref` = "at"
 **Вывод:** 2
 **Пояснение:** Две строки, содержащие "at" в качестве префикса: " **at** tention" и " **at** tendency".

**Пример 2:**

**Ввод:** words = ["leetcode","win","loops","success"], `pref` = "code"
 **Вывод:** 0
 **Пояснение:** Нет строк, содержащих "code" в качестве префикса.
