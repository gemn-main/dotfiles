```python
class Solution(object):
    def maxDepth(self, s):
        o = 0
        k = 0
        for i in range(len(s)):
            if k > o:
                o = k
            if s[i] == "(":
                k +=1
            if s[i] == ")":
                k -= 1
        return o
```

При наличии допустимой строки скобок s вернуть глубину вложенности . Глубина вложенности — это максимальное количество вложенных скобок. s

 

Пример 1:

Ввод: s = "(1+(2*3)+((8)/4))+1"

Выход: 3

Объяснение:

Цифра 8 находится внутри трех вложенных скобок в строке.

Пример 2:

Ввод: s = "(1)+((2))+(((3)))"

Выход: 3

Объяснение:

Цифра 3 находится внутри трех вложенных скобок в строке.

Пример 3:

Ввод: s = "()(())((()()))"

Выход: 3
