```python
class Solution(object):
    def maximumDifference(self, nums):
        if len(nums) <= 1:
            return nums
        h = -1
        for i in range(len(nums) - 1):
            k = nums[i]
            o = nums[i + 1:]
            for _ in range(len(o)):
                if o[_] - k > h:
                    h = o[_] - k
                    print(h)
        if h == 0:
            return -1
        return h
```
Для заданного целочисленного массива **с индексом 0**`nums` размера `n`найдите **максимальную разницу** между `nums[i]`и `nums[j]`(т.е. `nums[j] - nums[i]`), такую, что `0 <= i < j < n`и `nums[i] < nums[j]`.

Верните _максимальную **разницу** ._ Если такой нет `i`и `j`существует, верните `-1`.

**Пример 1:**

**Вход:** nums = [7, 1, 5 ,4]
 **Выход:** 4
 **Пояснение:** 
Максимальная разница возникает при i = 1 и j = 2, nums[j] - nums[i] = 5 - 1 = 4. 
Обратите внимание, что при i = 1 и j = 0 разница nums[j] - nums[i] = 7 - 1 = 6, но i > j, поэтому она недопустима.

**Пример 2:**

**Ввод:** nums = [9,4,3,2]
 **Вывод:** -1
 **Пояснение:** 
Не существует i и j таких, что i < j и nums[i] < nums[j].

**Пример 3:**

**Ввод:** nums = [ 1, 5, 2, 10 ]
 **Вывод:** 9
 **Пояснение:** 
Максимальная разница возникает при i = 0 и j = 3, nums[j] - nums[i] = 10 - 1 = 9.
