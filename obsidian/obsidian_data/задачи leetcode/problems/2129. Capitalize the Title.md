```python
class Solution(object):
    def capitalizeTitle(self, title):
        b = title.split(" ")
        o = ''
        for i in range(len(b)):
            if len(b[i]) <= 2:
                o += b[i].lower() + ' '
            else:
                o += b[i][0].upper() + b[i][1:].lower() + " "
                
        return o[:-1]
```
Вам дана строка, `title`состоящая из одного или нескольких слов, разделенных одним пробелом, где каждое слово состоит из английских букв. Сделайте строку **заглавной** , изменив заглавные буквы каждого слова таким образом, чтобы:

- Если длина слова составляет `1`или `2`букв, измените все буквы на строчные.
- В противном случае измените первую букву на заглавную, а остальные буквы — на строчные.

Верните _**заглавную**_ _букву_ `title` .

**Пример 1:**

**Ввод:** title = "capiTalIze the titleLe"
 **Вывод:** "Capitalize The Title"
 **Пояснение:** 
Поскольку все слова имеют длину не менее 3 символов, первая буква каждого слова заглавная, а остальные буквы — строчные.

**Пример 2:**

**Ввод:** title = "First leTTeR of EACH Word"
 **Вывод:** "First Letter of Each Word"
 **Пояснение:** 
Слово "of" имеет длину 2, поэтому оно все строчное. 
Длина остальных слов составляет не менее 3, поэтому первая буква каждого оставшегося слова заглавная, а остальные буквы — строчные.
