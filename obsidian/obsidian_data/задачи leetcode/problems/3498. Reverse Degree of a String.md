```cpp
class Solution {
public:
    int reverseDegree(string s) {
        int res = 0;
        for(int i =0; i < s.size(); i++){
            char b = s[i];
            int k = abs((b - 'a') - 26);
            res += k * (i+1);

       } 
        return res;
    }
};
```
Дана строка s, вычислите ее обратную степень .

Обратная степень рассчитывается следующим образом:

Для каждого символа умножьте его позицию в обратном алфавите ( 'a'= 26, 'b'= 25, ..., 'z'= 1) на его позицию в строке (индексированная на 1) .
Сложите эти произведения для всех символов в строке.
Верните обратную степень числа s.

 

Пример 1:

Ввод: s = "abc"

Вывод: 148

Объяснение:

Письмо	Индекс в обратном алфавите	Индекс в строке	Продукт
'a'	26	1	26
'b'	25	2	50
'c'	24	3	72
Обратная степень — 26 + 50 + 72 = 148.

Пример 2:

Ввод: s = "zaza"

Выход: 160

Объяснение:

Письмо	Индекс в обратном алфавите	Индекс в строке	Продукт
'z'	1	1	1
'a'	26	2	52
'z'	1	3	3
'a'	26	4	104
Обратная степень — 1 + 52 + 3 + 104 = 160.
