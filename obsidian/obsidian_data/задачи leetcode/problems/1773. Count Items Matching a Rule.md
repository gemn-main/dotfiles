```python
class Solution(object):
    def countMatches(self, items, ruleKey, ruleValue):
        o = 0
        for i in range(len(items)):
            if ruleKey == "type":
                if ruleValue == items[i][0]:
                    o += 1
            if ruleKey == "color":
                if ruleValue == items[i][1]:
                    o += 1
            if ruleKey == "name":
                if ruleValue == items[i][2]:
                    o += 1
        return o
```
Вам дан массив items, где каждый описывает тип, цвет и название элемента . Вам также дано правило, представленное двумя строками и .items[i] = [typei, colori, namei]ithruleKeyruleValue

ruleKey == "name"и .ruleValue == namei
Возвращает количество элементов, соответствующих заданному правилу .

 

Пример 1:

Ввод: items = ["phone","blue","pixel"],["computer","silver","lenovo"],["phone","gold","iphone"], ruleKey = "color", ruleValue = "silver"
 Вывод: 1
 Объяснение: Данному правилу соответствует только один элемент, а именно ["computer","silver","lenovo"].
Пример 2:
