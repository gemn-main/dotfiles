```python
class Solution(object):
    def licenseKeyFormatting(self, s, k):
        s = s.upper()
        s = s.replace('-', '')
        o = ''
        if len(s) <= k:
            return s
        a = len(s) % k
        b = s[a:]
        if a == 0:
            for i in range(len(b) // k):
                o += '-' + b[i * k:(i * k) + k]
            return o[1:]
        else:
            for i in range(len(b) // k):
                o += '-' + b[i * k:(i * k) + k]
            o = s[0:a] + o
            return o
```
Вам дается лицензионный ключ, представленный в виде строки `s`, которая состоит только из буквенно-цифровых символов и тире. Строка разделена на `n + 1`группы `n`тире. Вам также дается целое число `k`.

Мы хотим переформатировать строку `s`так, чтобы каждая группа содержала ровно `k`символов, за исключением первой группы, которая может быть короче `k`, но все равно должна содержать по крайней мере один символ. Кроме того, между двумя группами должен быть вставлен тире, и вам следует преобразовать все строчные буквы в заглавные.

Верните _переформатированный лицензионный ключ_.

**Пример 1:**

**Ввод:** s = "5F3Z-2e-9-w", k = 4
 **Вывод:** "5F3Z-2E9W"
 **Пояснение:** Строка s была разделена на две части, каждая часть имеет 4 символа. 
Обратите внимание, что два дополнительных тире не нужны и могут быть удалены.

**Пример 2:**

**Ввод:** s = "2-5g-3-J", k = 2
 **Вывод:** "2-5G-3J"
 **Пояснение:** Строка s была разделена на три части, каждая часть содержит 2 символа, за исключением первой части, поскольку она может быть короче, как указано выше.
