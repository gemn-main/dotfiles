```cpp
class Solution {
public:
    char nextGreatestLetter(vector<char>& letters, char target){
        char b = 127;
        for (char x: letters){
            if (x > target and x < b){
                b = x;
            }
        }
        if (b == 127)
            return letters[0];
        else
            return b;
    }
};
```

Вам дан массив символов `letters`, отсортированный в **неубывающем порядке** , и символ `target`. В . Есть **как минимум два различных** символа `letters`.

Возвращает _наименьший символ в ,_ `letters`_который лексикографически больше_`target` . Если такой символ не существует, возвращает первый символ в `letters`.

**Пример 1:**

**Ввод:** буквы = ["c","f","j"], цель = "a"
 **Вывод:** "c"
 **Пояснение:** Наименьший символ, который лексикографически больше, чем "a" в буквах, - это "c".

**Пример 2:**

**Ввод:** буквы = ["c","f","j"], цель = "c"
 **Вывод:** "f"
 **Пояснение:** Наименьший символ, который лексикографически больше "c" в буквах, - это "f".

**Пример 3:**

**Ввод:** letters = ["x","x","y","y"], цель = "z"
 **Вывод:** "x"
 **Пояснение:** в letters нет символов, которые лексикографически больше, чем 'z', поэтому мы возвращаем letters[0].
