```cpp
class Solution(object):
    def countNegatives(self, grid):
        o = 0
        for i in range(len(grid)):
            for j in range(len(grid[i])):
                if grid[i][j] < 0:
                    o+=1
        return o
```
Дана m x nматрица grid, отсортированная в невозрастающем порядке как по строкам, так и по столбцам. Верните количество отрицательных чисел в grid .

 

Пример 1:

Вход: сетка = [4,3,2,-1],
			 [3,2,1,-1],
			 [1,1,-1,-2],
			 [-1,-1,-2,-3]
 Выход: 8
 Пояснение: В матрице 8 отрицательных чисел.
Пример 2:

Вход: сетка = [3,2],[1,0]
 Выход: 0
