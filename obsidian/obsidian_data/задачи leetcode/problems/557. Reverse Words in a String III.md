```python
class Solution(object):
    def reverseWords(self, s):
        b = s.split(' ')
        a = ''
        for i in range(len(b)):
            a = a + b[i][::-1] + ' '
        return a[:-1]
```
Для заданной строки `s`поменяйте порядок символов в каждом слове предложения на обратный, сохраняя при этом пробелы и начальный порядок слов.

**Example 1:**

**Input:** s = "Let's take LeetCode contest"
**Output:** "s'teL ekat edoCteeL tsetnoc"

**Example 2:**

**Input:** s = "Mr Ding"
**Output:** "rM gniD"
