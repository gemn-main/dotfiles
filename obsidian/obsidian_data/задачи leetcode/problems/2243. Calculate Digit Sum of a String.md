```python
class Solution(object):
    def digitSum(self, s, k):
        b = True
        while True:
            a = []
            if len(s) <= k:
                return s
            for i in range(0, len(s), k):
                a.append(s[i:i+k])
            r = ''
            for i in range(len(a)):
                o = 0
                for j in range(len(a[i])):
                    o += int(a[i][j])
                r += str(o)
            s= r
```
Вам дана строка, `s`состоящая из цифр и целого числа `k`.

Раунд можно считать завершенным **,** если длина `s`больше `k`. В одном раунде выполните следующие действия:

1. **Разделите** `s` на **последовательные группы** размера `k`так, чтобы первые `k`символы были в первой группе, следующие `k`символы — во второй группе и т. д. **Обратите внимание** , что размер последней группы может быть меньше `k`.
2. **Заменить** каждую группу `s`на строку, представляющую сумму всех ее цифр. Например, `"346"`заменяется на `"13"`because `3 + 4 + 6 = 13`.
3. **Объедините** последовательные группы вместе, чтобы сформировать новую строку. Если длина строки больше `k`, повторите с шага `1`.

Возвращайтесь `s` _после завершения всех раундов_ .

**Пример 1:**

**Вход:** s = "11111222223", k = 3
 **Выход:** "135"
 **Пояснение:** - Для первого раунда мы делим s на группы размером 3: "111", "112", "222" и "23". ​​​​​Затем мы вычисляем сумму цифр каждой группы: 1 + 1 + 1 = 3, 1 + 1 + 2 = 4, 2 + 2 + 2 = 6 и 2 + 3 = 5.   Таким образом, s становится "3" + "4" + "6" + "5" = "3465" после первого раунда. - Для второго раунда мы делим s на "346" и "5".   Затем мы вычисляем сумму цифр каждой группы: 3 + 4 + 6 = 13, 5 = 5.   Таким образом, s становится "13" + "5" = "135" после второго раунда. 
Теперь s.length <= k, поэтому мы возвращаем "135" в качестве ответа.

**Пример 2:**

**Вход:** s = "00000000", k = 3
 **Выход:** "000"
 **Объяснение:** Мы делим s на "000", "000" и "00". Затем мы вычисляем сумму цифр каждой группы: 0 + 0 + 0 = 0, 0 + 0 + 0 = 0 и 0 + 0 = 0. 
s становится "0" + "0" + "0" = "000", длина которого равна k, поэтому мы возвращаем "000".
