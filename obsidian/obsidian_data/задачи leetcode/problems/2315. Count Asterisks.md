```python
class Solution(object):
    def countAsterisks(self, s):
        o = 0
        k = 0
        for i in range(len(s)):
            if s[i] == "|":
                if k == 1:
                    k = 0
                else:
                    k = 1
            else:
                if s[i] == "*":
                    if k == 0:
                        o +=1
        return o
```
Вам дана строка `s`, где каждые **две** последовательные вертикальные полосы `'|'`сгруппированы в **пару** . Другими словами, 1- я и 2 -я `'|'` составляют пару, 3 -я и 4 -я `'|'` составляют пару и т. д.

Верните _количество_ `'*'`_в_ `s`_, **исключая** между_ `'*'`_каждой парой_`'|'` .

**Обратите внимание** , что каждый из них `'|'`будет принадлежать **только** одной паре.

**Пример 1:**

**Вход:** s = "l|*e*et|c**o|*de|"
 **Выход:** 2
 **Пояснение:** Рассматриваемые символы подчеркнуты: " l |*e*et| c**o |*de|". Символы между первым и вторым '|' исключаются из ответа. Также из ответа исключаются символы между третьим и четвертым '|'. 
Рассматриваются 2 звездочки. Поэтому мы возвращаем 2.

**Пример 2:**

**Ввод:** s = "iamprogrammer"
 **Вывод:** 0
 **Пояснение:** В этом примере в s нет звездочек. Поэтому мы возвращаем 0.

**Пример 3:**

**Вход:** s = "yo|uar|e**|b|e***au|tifu|l"
 **Выход:** 5
 **Пояснение:** Рассматриваемые символы подчеркнуты: " yo |uar| e** |b| e***au |tifu| l ". Рассматривается 5 звездочек. Поэтому возвращаем 5.
