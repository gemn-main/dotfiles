```python
class Solution(object):
    def buddyStrings(self, s, goal):
        s = list(s)
        goal = list(goal)
        if len(s) != len(goal): return False
        if len(s) == 1:
            return False
        if s == goal:
            if len(set(s)) <= len(s) - 2:
                return True
            if len(set(s)) == 1 and len(set(goal)) == 1:
                return True
        o = []
        for i in range(len(s)):
            if s[i] != goal[i]:
                o.append(i)
        if len(o) != 2:
            return False
        b = s[o[0]]
        s[o[0]] = s[o[1]]
        s[o[1]] = b
        if s != goal:
            return False  
        return True  
```
Даны две строки sи goal, верните , trueесли вы можете поменять местами две буквы sтак, чтобы результат был равен goal, в противном случае верните false.

Перестановка букв определяется как взятие двух индексов iи j(с индексом 0) таких, что i != jи перестановка символов в s[i]и s[j].

Например, замена индексов 0и 2приводит "abcd"к результату "cbad".
 

Пример 1:

Ввод: s = "ab", цель = "ba"
 Вывод: true
 Пояснение: Вы можете поменять местами s[0] = 'a' и s[1] = 'b', чтобы получить "ba", что равно цели.
Пример 2:

Ввод: s = "ab", цель = "ab"
 Вывод: false
 Пояснение: Единственные буквы, которые вы можете поменять местами, это s[0] = 'a' и s[1] = 'b', что дает "ba" != цель.
Пример 3:

Ввод: s = "aa", цель = "aa"
 Вывод: true
 Пояснение: Вы можете поменять местами s[0] = 'a' и s[1] = 'a', чтобы получить "aa", что равно цели.
