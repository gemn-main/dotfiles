```python
class Solution(object):
    def satisfiesConditions(self, grid):
        for i in range(len(grid)):
            for j in range(len(grid[i])-1):
                if grid[i][j] == grid[i][j+1]:
                    return False
        for i in range(len(grid)-1):
            for j in range(len(grid[i])):
                if grid[i][j] != grid[i+1][j]:
                    return False
        return True           
```
Вам дана двумерная матрица gridразмером m x n. Вам нужно проверить, grid[i][j]является ли каждая ячейка:

Равен ячейке под ней, т.е. grid  [i]   [j] == grid [i + 1]  [j] (если она существует).
Отличается от ячейки справа, т.е. grid[i]  [j] != grid  [i]   [j + 1] (если она существует).
Верните, trueесли все ячейки удовлетворяют этим условиям, в противном случае верните false.

 

Пример 1:

Ввод: сетка = [1,0,2],[1,0,2]

Вывод: истина

Объяснение:



Все ячейки сетки удовлетворяют условиям.

Пример 2:

Ввод: сетка = [1,1,1],[0,0,0]

Вывод: ложь

Объяснение:



Все ячейки в первой строке равны.

Пример 3:

Ввод: сетка = [1],[2],[3]

Вывод: ложь

Объяснение:



Ячейки в первом столбце имеют разные значения.
