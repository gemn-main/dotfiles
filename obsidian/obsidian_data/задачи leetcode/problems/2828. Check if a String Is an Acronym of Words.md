```python
class Solution(object):
    def isAcronym(self, words, s):
        if len(words) != len(s):
            return False
        for i in range(len(words)):
            if words[i][0] != s[i]:
                return False
        return True
```

Дан массив строк wordsи строка s, определите, sявляется ли она аббревиатурой слов.

Строка sсчитается аббревиатурой, wordsесли ее можно образовать путем конкатенации первых символов каждой строки в words порядке . Например, "ab"можно образовать из ["apple", "banana"], но нельзя образовать из ["bear", "aardvark"].

Верните true, если sэто аббревиатура words, и falseв противном случае.

 

Пример 1:

Ввод: words = ["alice","bob","charlie"], s = "abc"
 Вывод: true
 Пояснение: Первые символы в словах "alice", "bob" и "charlie" - это 'a', 'b' и 'c' соответственно. Следовательно, s = "abc" - это аббревиатура. 
Пример 2:

Ввод: words = ["an","apple"], s = "a"
 Вывод: false
 Пояснение: Первые символы в словах "an" и "apple" — это "a" и "a" соответственно.
Аббревиатура, образованная путем объединения этих символов, — «аа».
Следовательно, s = "a" не является аббревиатурой.
Пример 3:

Ввод: words = ["never","gonna","give","up","on","you"], s = "ngguoy"
 Вывод: true
 Пояснение: Объединяя первые символы слов в массиве, мы получаем строку "ngguoy".
Следовательно, s = "ngguoy" — это аббревиатура.
