```cpp
class Solution(object):
    def canMakeSquare(self, grid):
        one =   [grid[0][0], grid[0][1], grid[1][0], grid[1][1]]
        two =   [grid[0][1], grid[0][2], grid[1][1], grid[1][2]]
        three = [grid[1][0], grid[1][1], grid[2][0], grid[2][1]]
        four =  [grid[1][1], grid[1][2], grid[2][1], grid[2][2]]
        use = [one, two, three, four]
        for i in range(4):
            b = use[i].count('W')
            if b == 0 or b == 3 or b == 1 or b == 4:
                return True
        return False
```
Вам дана двумерная матрица `grid`размера, `3 x 3`состоящая только из символов `'B'`и `'W'`. Символ `'W'`представляет белый цвет , а символ `'B'`представляет черный цвет .

Ваша задача — изменить цвет **не более чем одной** ячейки так, чтобы в матрице образовался `2 x 2`квадрат, все ячейки которого имеют один цвет.

Верните, `true`если возможно создать `2 x 2` квадрат того же цвета, в противном случае верните `false`.

**Пример 1:**

**Ввод:** сетка = ["B","W","B"],
			  ["B","W","W"],
			  ["B","W","B"]

**Вывод:** истина

**Объяснение:**

Это можно сделать, изменив цвет `grid[0][2]`.

**Пример 2:**

**Ввод:** сетка = ["B","W","B"],
              ["W","B","W"],
              ["B","W","B"]

**Вывод:** ложь

**Объяснение:**

Этого нельзя добиться, изменив максимум одну ячейку.

**Пример 3:**

**Ввод:** сетка = ["B","W","B"],
			  ["B","W","W"],
              ["B","W","W"]

**Вывод:** истина

**Объяснение:**

В `grid` уже есть `2 x 2` квадрат того же цвета.
