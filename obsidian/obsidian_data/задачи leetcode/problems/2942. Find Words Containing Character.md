```python
class Solution(object):
    def findWordsContaining(self, words, x):
        o = []
        for i in range(len(words)):
            if x in words[i]:
                o.append(i)
        return o
```
Вам дан индексированный 0 массив строк wordsи символ x.

Возвращает массив индексов, представляющих слова, содержащие символx .

Обратите внимание , что возвращаемый массив может иметь любой порядок.

 

Пример 1:

Вход: words = ["leet","code"], x = "e"
 Выход: [0,1]
 Пояснение: "e" встречается в обоих словах: "l ee t" и "code e ". Следовательно, мы возвращаем индексы 0 и 1.
Пример 2:

Вход: words = ["abc","bcd","aaaa","cbc"], x = "a"
 Выход: [0,2]
 Пояснение: "a" встречается в " a bc" и " aaaa ". Следовательно, мы возвращаем индексы 0 и 2.
Пример 3:

Вход: words = ["abc","bcd","aaaa","cbc"], x = "z"
 Выход: []
 Пояснение: "z" не встречается ни в одном из слов. Следовательно, мы возвращаем пустой массив.
