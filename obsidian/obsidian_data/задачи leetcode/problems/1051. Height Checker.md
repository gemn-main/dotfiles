```python
class Solution(object):
    def heightChecker(self, heights):
        o = 0
        b = heights[:]
        b.sort()
        for i in range(len(heights)):
            if b[i] != heights[i]:
                o+=1
        return o
```
Школа пытается сделать ежегодное фото всех учеников. Учащихся просят встать в одну линию в неубывающем порядке по росту. Пусть этот порядок будет представлен целочисленным массивом, expectedгде expected[i]— ожидаемый рост ученика в очереди.ith

Вам дан целочисленный массив, heightsпредставляющий текущий порядок расположения учеников. Каждое число heights[i]— это рост ученика в очереди ( индексация начинается с 0 ).ith

Верните количество индексов , гдеheights[i] != expected[i] .

 Выход: 3

Ожидается: [1,1, 1 ,2, 3 , 4 ] 
Индексы 2, 4 и 5 не совпадают.
