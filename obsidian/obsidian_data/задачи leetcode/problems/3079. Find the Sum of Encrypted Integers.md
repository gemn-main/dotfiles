```python
class Solution(object):
    def sumOfEncryptedInt(self, nums):
        def encrypt(x):
            x = str(x)
            o = 0 
            for i in x:
                int_i = int(i)
                if int_i > o:
                    o = int_i
            return int(len(x) * str(o))
        k = 0
        for i in range(len(nums)):
            k += encrypt(nums[i])
        return k
```
Вам дан массив целых чисел nums, содержащий положительные целые числа. Мы определяем функцию encrypt, которая encrypt(x)заменяет каждую цифру в xнаибольшей цифрой в x. Например, encrypt(523) = 555и encrypt(213) = 333.

Верните сумму зашифрованных элементов .

 

Пример 1:

Ввод: числа = [1,2,3]

Выход: 6

Пояснение: Зашифрованные элементы —  [1,2,3]. Сумма зашифрованных элементов — 1 + 2 + 3 == 6.

Выход: 66

Пояснение: Зашифрованные элементы — [11,22,33]. Сумма зашифрованных элементов — 11 + 22 + 33 == 66.
