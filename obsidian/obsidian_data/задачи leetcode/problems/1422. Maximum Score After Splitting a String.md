```python
class Solution(object):
    def maxScore(self, s):
        def zeros(z):
            b = 0
            for i in range(len(z)):
                if z[i] == '0':
                    b += 1
            return b
        def ones(o):
            p  = 0
            for i in range(len(o)):
                if o[i] == '1':
                    p += 1
            return p
        k = 0
        for i in range(1,len(s)):
            o = zeros(s[:i]) + ones(s[i:]) 
            if k < o:
                k = o
        return k
```

Дана строка s из нулей и единиц, вернуть максимальный балл после разбиения строки на две непустые подстроки (т. е. левую подстроку и правую подстроку).

Результатом разделения строки является количество нулей в левой подстроке плюс количество единиц в правой подстроке.

 

Пример 1:

Вход: s = "011101"
 Выход: 5 
 Пояснение:  
Все возможные способы разбиения s на две непустые подстроки: 
left = "0" и right = "11101", счет = 1 + 4 = 5 
left = "01" и right = "1101", счет = 1 + 3 = 4 
left = "011" и right = "101", счет = 1 + 2 = 3 
left = "0111" и right = "01", счет = 1 + 1 = 2 
left = "01110" и right = "1", счет = 2 + 1 = 3
Пример 2:

Ввод: s = «00111»
 Вывод: 5
 Пояснение: Когда слева = «00», а справа = «111», мы получаем максимальный балл = 2 + 3 = 5
Пример 3:

Ввод: s = "1111"
 Вывод: 3
