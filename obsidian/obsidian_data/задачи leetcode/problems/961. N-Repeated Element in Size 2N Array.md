```python
class Solution(object):
    def repeatedNTimes(self, nums):
        o = []
        for i in nums:
            if i not in o:
                o.append(i)
            else:
                return i
            
```

Вам дан целочисленный массив numsсо следующими свойствами:

nums.length == 2 * n.
numsсодержит n + 1 уникальные элементы.
Ровно один элемент numsповторяется nраз.
Верните элемент, который повторяется nраз .

 

Пример 1:

Ввод: числа = [1,2,3,3]
 Вывод: 3
Пример 2:

Ввод: числа = [2,1,2,5,3,2]
 Вывод: 2
Пример 3:

Ввод: числа = [5,1,5,2,5,3,5,4]
 Вывод: 5
