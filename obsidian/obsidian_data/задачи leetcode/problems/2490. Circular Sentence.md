```python
class Solution(object):  
    def isCircularSentence(self, sentence):  
        b = sentence.split(' ')  
        x = 0  
        if len(b) == 1:  
            if b[0][0] == b[-1][-1]:  
                print("1")  
                return True  
            else:  
                return False  
        if b[0][0] == b[-1][-1]:  
            for i in range(len(b) - 1):  
                if b[i][-1] == b[i + 1][0]:  
                    x = x + 1  
  
        return x == len(b) - 1
```

Предложение является **циклическим,** если:

- Последний символ слова равен первому символу следующего слова.
- Последний символ последнего слова равен первому символу первого слова.
- 
Например, `"leetcode exercises sound delightful"`, `"eetcode"`, `"leetcode eats soul"` все являются циклическими предложениями. Однако `"Leetcode is cool"`и **не являются** циклическими предложениями.`"happy Leetcode"``"Leetcode"``"I like Leetcode"` 

Дана строка `sentence`, вернуть , `true`_если она циклическая_ . В противном случае вернуть `false`.
