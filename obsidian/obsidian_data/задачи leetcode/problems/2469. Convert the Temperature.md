```python
class Solution(object):
    def convertTemperature(self, celsius):
        o = []
        o.append(celsius + 273.15)
        o.append((celsius * 1.8) + 32)
        return o
```
Вам дано неотрицательное число с плавающей запятой, округленное до двух знаков после запятой celsius, которое обозначает температуру в градусах Цельсия .

Вам следует преобразовать градусы Цельсия в градусы Кельвина и Фаренгейта и вернуть результат в виде массива ans = [kelvin, fahrenheit].

Верните массив ans. Ответы в пределах фактического ответа будут приняты.10-5

Обратите внимание, что:

Kelvin = Celsius + 273.15
Fahrenheit = Celsius * 1.80 + 32.00
 

Пример 1:

Ввод: градус Цельсия = 36,50
 Вывод: [309,65000,97,70000]
 Объяснение: Температура при 36,50 градусах Цельсия, преобразованная в градусы Кельвина, составляет 309,65, а преобразованная в градусы Фаренгейта — 97,70.
Пример 2:

Ввод: градус Цельсия = 122,11
 Вывод: [395,26000,251,79800]
 Объяснение: Температура при 122,11 градусах Цельсия, преобразованная в градусы Кельвина, составляет 395,26, а преобразованная в градусы Фаренгейта — 251,798.
