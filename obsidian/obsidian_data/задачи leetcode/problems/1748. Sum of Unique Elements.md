```python
class Solution(object):
    def sumOfUnique(self, nums):
        if len(nums) == 1:
            return nums[0]
        o = []
        a = []
        for i in range(len(nums)):
            if nums[i] in o:
                a.append(nums[i])
                o.remove(nums[i])
            if nums[i] in a:
                pass
            else:
                o.append(nums[i])
        return sum(o)
```
Вам дан целочисленный массив `nums`. Уникальными элементами массива являются элементы, которые встречаются в массиве **ровно один раз .**

Верните _сумму всех уникальных **элементов**_`nums` .

**Пример 1:**

**Ввод:** nums = [1,2,3,2]
 **Вывод:** 4
 **Пояснение:** Уникальные элементы — [1,3], а сумма — 4.

**Пример 2:**

**Ввод:** nums = [1,1,1,1,1]
 **Вывод:** 0
 **Пояснение:** Уникальных элементов нет, а сумма равна 0.

**Пример 3:**

**Ввод:** nums = [1,2,3,4,5]
 **Вывод:** 15
 **Пояснение:** Уникальные элементы — [1,2,3,4,5], а сумма равна 15.
