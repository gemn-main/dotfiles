```python
class Solution(object):
    def secondHighest(self, s):
        k = []
        for i in range(len(s)):
            if s[i].isdigit():
                if int(s[i]) not in k:
                    k.append(int(s[i]))
        k.sort()
        if len(k) < 2:
            return -1
        else:
            return k[-2]
```

Для заданной буквенно-цифровой строки sвернуть вторую по величине числовую цифру, встречающуюся в s, или, -1если она не существует , вернуть .

Буквенно -цифровая строка — это строка, состоящая из строчных английских букв и цифр.

 

Пример 1:

Ввод: s = "dfa12321afd"
 Вывод: 2
 Пояснение: Цифры, которые появляются в s, это [1, 2, 3]. Вторая по величине цифра - 2.
Пример 2:

Ввод: s = "abc1111"
 Вывод: -1
 Пояснение: Цифры, которые появляются в s, это [1]. Второй по величине цифры нет.
