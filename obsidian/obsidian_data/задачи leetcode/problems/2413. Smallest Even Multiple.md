```python
class Solution(object):
    def smallestEvenMultiple(self, n):
        if n % 2 == 0:
            return n
        else:
            return n*2
```
Для заданного положительного целого числа nвернуть наименьшее положительное целое число, кратное как 2 иn .
 

Пример 1:

Вход: n = 5
 Выход: 10
 Пояснение: Наименьшее кратное 5 и 2 — 10.
Пример 2:

Вход: n = 6
 Выход: 6
 Пояснение: Наименьшее кратное 6 и 2 — это 6. Обратите внимание, что число является кратным самому себе.
