```python
def loli(n):  
    o = [n]  
    while True:  
        if n == 1:  
            break  
        if n % 2 == 0:  
            n = n // 2  
            o.append(n)  
        else:  
            n = n * 3 + 1  
            o.append(n)  
    return o  
  
n = 3  
print(loli(n))

```
- Принимает в качестве входных данных положительное целое число *n* 
- Если *n* четное, алгоритм делит его на два, и если *n* нечетное, алгоритм умножает его на три и прибавляет один.
