```python
class Solution(object):
    def countConsistentStrings(self, allowed, words):
        o = 0
        for i in words:
            k = 0
            for j in range(len(i)):
                if i[j] not in allowed:
                    k = 1
            if k != 1:
                o +=1
        return o
```

Вам дана строка, allowedсостоящая из отдельных символов и массив строк words. Строка является согласованной, если все символы в строке появляются в строке allowed.

Верните количество согласованных строк в массивеwords .

 

Пример 1:

Ввод: разрешено = "ab", слова = ["ad","bd","aaab","baa","badab"]
 Вывод: 2
 Пояснение: Строки "aaab" и "baa" согласованы, поскольку содержат только символы "a" и "b".
Пример 2:

Ввод: разрешено = "abc", слова = ["a","b","c","ab","ac","bc","abc"]
 Вывод: 7
 Пояснение: Все строки согласованы.
Пример 3:

Ввод: разрешено = "cad", слова = ["cc","acd","b","ba","bac","bad","ac","d"]
 Вывод: 4
 Пояснение: Строки "cc", "acd", "ac" и "d" согласованы.
