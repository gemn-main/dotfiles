```cpp
class Solution {
public:
    vector<int> findIndices(vector<int>& nums, int indexDifference, int valueDifference) {
        int n = nums.size();
        if (n >= (indexDifference +1))
        for(int i =0; i <= (n-indexDifference-1); i++){
            for(int j = i+indexDifference; j<=n-1; j++){
                cout << i << j << endl;
                if (abs(nums[i] - nums[j]) >= valueDifference)
                    return {i, j};
            }
        }
        return {-1, -1};
    }
};
```

Вам дан целочисленный массив с индексом 0,nums имеющий длину n, целое число indexDifferenceи целое число valueDifference.

Ваша задача — найти два индекса iи j, оба в диапазоне [0, n - 1], которые удовлетворяют следующим условиям:

abs(i - j) >= indexDifference, и
abs(nums[i] - nums[j]) >= valueDifference
Верните целочисленный массив answer , где answer = [i, j] если таких индексов два , и answer = [-1, -1] в противном случае . Если для двух индексов есть несколько вариантов, верните любой из них .

Примечание: i и jмогут быть равны .

 

Пример 1:

Ввод: nums = [5,1,4,1], indexDifference = 2, valueDifference = 4
 Вывод: [0,3]
 Пояснение: В этом примере можно выбрать i = 0 и j = 3. 
abs(0 - 3) >= 2 и abs(nums[0] - nums[3]) >= 4. 
Следовательно, допустимый ответ — [0,3]. 
[3,0] также является допустимым ответом.
Пример 2:

Ввод: nums = [2,1], indexDifference = 0, valueDifference = 0
 Вывод: [0,0]
 Пояснение: В этом примере можно выбрать i = 0 и j = 0. 
abs(0 - 0) >= 0 и abs(nums[0] - nums[0]) >= 0. 
Следовательно, допустимый ответ - [0,0]. 
Другие допустимые ответы - [0,1], [1,0] и [1,1].
Пример 3:

Вход: nums = [1,2,3], indexDifference = 2, valueDifference = 4
 Выход: [-1,-1]
 Пояснение: В этом примере можно показать, что невозможно найти два индекса, удовлетворяющих обоим условиям. 
Следовательно, возвращается [-1,-1].
