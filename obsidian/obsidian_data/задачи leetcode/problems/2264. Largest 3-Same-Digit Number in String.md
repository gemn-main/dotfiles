```python
class Solution(object):
    def largestGoodInteger(self, num):
        o = '-1'
        for i in range(len(num)-2):
            newstr = num[i] + num[i+1] + num[i+2]
            if newstr[0] == newstr[1] == newstr[2]:
                if int(newstr) >= int(o):
                    o = newstr
        if o == "-1":
            return ""
        return o
```
Вам дана строка, numпредставляющая большое целое число. Целое число является хорошим, если оно удовлетворяет следующим условиям:

Это подстрока длиной .num​3
Он состоит только из одной уникальной цифры.
Возвращает максимальное хорошее целое число в виде строки или пустую строку, ""если такое целое число не существует .

Примечание:

Подстрока — это непрерывная последовательность символов внутри строки.
В начале могут быть нулиnum или целое число.
 

Пример 1:

Ввод: num = "6 777 133339"
 Вывод: "777"
 Пояснение: Есть два различных хороших целых числа: "777" и "333". 
"777" — наибольшее, поэтому мы возвращаем "777".
Пример 2:

Ввод: num = "23 000 19"
 Вывод: "000"
 Пояснение: "000" — единственное подходящее целое число.
Пример 3:

Ввод: num = "42352338"
 Вывод: ""
 Объяснение: Ни одна подстрока длины 3 не состоит только из одной уникальной цифры. Следовательно, хороших целых чисел нет.
