```python
class Solution(object):
    def arrayStringsAreEqual(self, word1, word2):
        o = ''
        a = ''
        for i in range(len(word1)):
            o += word1[i]
        for i in range(len(word2)):
            a += word2[i]
            
        if a == o:
            return True
        else:
            return False
```
Даны два строковых массива `word1`и `word2`, вернуть , _если два массива_ _**представляют**_ _одну и ту же строку, и_ _в противном случае._ `true``false`

Строка **представляется** массивом, если элементы массива, объединенные **по порядку,** образуют строку.

**Пример 1:**

**Ввод:** word1 = ["ab", "c"], word2 = ["a", "bc"]
 **Вывод:** true
 **Пояснение:** 
word1 представляет строку "ab" + "c" -> "abc" word2 представляет строку "a" + "bc" -> "abc" 
Строки одинаковы, поэтому возвращаем true.

**Пример 2:**

**Ввод:** word1 = ["a", "cb"], word2 = ["ab", "c"]
 **Вывод:** false

**Пример 3:**

**Ввод:** word1 = ["abc", "d", "defg"], word2 = ["abcddefg"]
 **Вывод:** true
