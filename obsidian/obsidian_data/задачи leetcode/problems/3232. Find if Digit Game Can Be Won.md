```python
class Solution(object):
    def canAliceWin(self, nums):
        o = 0
        a = 0
        for i in range(len(nums)):
            num = nums[i]
            if num >= 10:
                o += num
            else:
                a += num
        return a != o
```
Вам дан массив **положительных** целых чисел `nums`.

Алиса и Боб играют в игру. В игре Алиса может выбрать **либо** все однозначные числа, либо все двузначные числа из `nums`, а остальные числа отдаются Бобу. Алиса выигрывает, если сумма ее чисел **строго больше** суммы чисел Боба.

Верните, `true`если Алиса может выиграть в этой игре, в противном случае верните `false`.

**Пример 1:**

**Ввод:** числа = [1,2,3,4,10]

**Вывод:** ложь

**Объяснение:**

Алиса не может выиграть, выбрав однозначные или двузначные числа.

**Пример 2:**

**Ввод:** числа = [1,2,3,4,5,14]

**Вывод:** истина

**Объяснение:**

Алиса может выиграть, выбрав однозначные числа, сумма которых равна 15.

**Пример 3:**

**Ввод:** числа = [5,5,5,25]

**Вывод:** истина

**Объяснение:**

Алиса может выиграть, выбрав двузначные числа, сумма которых равна 25.
