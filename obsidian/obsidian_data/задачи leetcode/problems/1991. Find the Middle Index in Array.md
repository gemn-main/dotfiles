```python
class Solution(object):
    def findMiddleIndex(self, nums):
        for i in range(len(nums)):
            if sum(nums[:i]) == sum(nums[i+1:]):
                return i
        return -1
```
Дан целочисленный массив **с индексом 0.**`nums` Найдите самый **левый элемент** `middleIndex` (т.е. наименьший среди всех возможных).

A `middleIndex`— это индекс, где `nums[0] + nums[1] + ... + nums[middleIndex-1] == nums[middleIndex+1] + nums[middleIndex+2] + ... + nums[nums.length-1]`.

Если `middleIndex == 0`, то сумма в левой части считается равной `0`. Аналогично, если `middleIndex == nums.length - 1`, то сумма в правой части считается равной `0`.

_Возвращает самый **левый**_ `middleIndex` _индекс, удовлетворяющий условию, или,_ `-1`_если такого индекса нет_ , возвращает .

**Пример 1:**

**Ввод:** nums = [2,3,-1, 8 ,4]
 **Вывод:** 3
 **Пояснение:** Сумма чисел до индекса 3 равна: 2 + 3 + -1 = 4 
Сумма чисел после индекса 3 равна: 4 = 4

**Пример 2:**

**Ввод:** nums = [1,-1, 4 ]
 **Вывод:** 2
 **Пояснение:** Сумма чисел до индекса 2 равна: 1 + -1 = 0 
Сумма чисел после индекса 2 равна: 0

**Пример 3:**

**Ввод:** nums = [2,5]
 **Вывод:** -1
 **Пояснение:** Отсутствует допустимый middleIndex.
