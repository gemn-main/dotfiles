```python
class Solution(object):
    def getConcatenation(self, nums):
        a = nums + nums
        return a
```
Дан целочисленный массив `nums`длины `n`, вы хотите создать массив `a` длины `2n` где `a[i] == nums[i]` и `a[i + n] == nums[i]`для `0 <= i < n`( **с индексом 0** ).

В частности, `a` это **объединение** двух `nums`массивов.

Верните _массив_ `a`

**Example 1:**

**Input:** nums = [1,2,1]
**Output:** [1,2,1,1,2,1]
**Explanation:** The array ans is formed as follows:
- ans = [nums[0],nums[1],nums[2],nums[0],nums[1],nums[2]]
- ans = [1,2,1,1,2,1]

**Example 2:**

**Input:** nums = [1,3,2,1]
**Output:** [1,3,2,1,1,3,2,1]
**Explanation:** The array ans is formed as follows:
- ans = [nums[0],nums[1],nums[2],nums[3],nums[0],nums[1],nums[2],nums[3]]
- ans = [1,3,2,1,1,3,2,1]
