```python
class Solution(object):
    def distinctIntegers(self, n):
        o = [n]
        i = 0
        b = [n]
        while i <= (10 ** 9):
            a = 0
            new_b = []
            for i in range(len(b)):
                for j in range(1,b[i]):
                    if b[i] % j == 1:
                        if j not in o:
                            a += 1
                            o.append(j)
                            new_b.append(j)
            i += 1
            if a == 0:
                return len(o)
            b = new_b
```
Вам дано положительное целое число `n`, которое изначально размещено на доске. Каждый день, в течение дней, вы выполняете следующую процедуру:`109`

- Для каждого числа, `x`представленного на доске, найдите все числа `1 <= i <= n`такие, что `x % i == 1`.
- Затем разместите эти числа на доске.

Верните _количество **различных** целых чисел, присутствующих на доске по_ _истечении указанного количества дней_ .`109`

**Примечание:**

- После того, как число будет помещено на доску, оно останется там до конца.
- `%` обозначает операцию по модулю. Например  , `14 % 3`.`2`

**Пример 1:**

**Вход:** n = 5
 **Выход:** 4
 **Объяснение:** Изначально на доске присутствует 5. На следующий день будут добавлены 2 и 4, так как 5 % 2 == 1 и 5 % 4 == 1. После этого дня на доску будет добавлено 3, так как 4 % 3 == 1. 
По истечении миллиарда дней на доске будут стоять различные числа 2, 3, 4 и 5.

**Пример 2:**

**Вход:** n = 3
 **Выход:** 2
 **Объяснение:** Поскольку 3 % 2 == 1, на доску будет добавлено 2. 
Через миллиард дней на доске останутся только два различных числа — 2 и 3.
