```python
class Solution(object):
    def kLengthApart(self, nums, k):
        if k == 0:
            return True
        for i in range(len(nums)):
            if nums[i] == 1:
                if i == len(nums)-1:
                    pass
                else:
                    o = 0
                    for j in range(i+1,len(nums)):
                        if nums[j] == 0:
                            o += 1
                        else:
                            if o < k:
                                return False
                            else:
                                break
        return True         
```

Дан двоичный массив `nums`и целое число `k`, вернуть , `true`_если все_ `1`_элементы находятся на_ `k`_расстоянии не менее одной позиции друг от друга, в противном случае вернуть_`false` .

**Пример 1:**

![](https://assets.leetcode.com/uploads/2020/04/15/sample_1_1791.png)

**Ввод:** nums = [1,0,0,0,1,0,0,1], k = 2
 **Вывод:** true
 **Пояснение:** Каждая из единиц находится на расстоянии не менее 2 позиций друг от друга.

**Пример 2:**

![](https://assets.leetcode.com/uploads/2020/04/15/sample_2_1791.png)

**Ввод:** nums = [1,0,0,1,0,1], k = 2
 **Вывод:** false
 **Пояснение:** Вторая 1 и третья 1 отстоят друг от друга всего на единицу.
