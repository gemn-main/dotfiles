```python
class Solution(object):
    def checkOnesSegment(self, s):
        b = 0
        flack = False
        for i in range(len(s)):
            if s[i] == "1":
                if flack == False:
                    flack = True
                    b += 1
            else:
                flack = False
        return b <= 1
```
Если задана двоичная строка s без начальных нулей , вернуть, еслиtrue содержит не более одного непрерывного сегмента единиц . В противном случае вернуть .sfalse

 

Пример 1:

Ввод: s = "1001"
 Вывод: false
 Пояснение: Единицы не образуют непрерывный сегмент.
Пример 2:

Вход: s = "110"
 Выход: true
