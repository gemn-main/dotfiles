```python
class Solution(object):
    def maxProduct(self, nums):
        nums.sort()
        return (nums[-1] -1) * (nums[-2] -1)
```
Учитывая массив целых чисел `nums`, вы выберете два разных индекса `i`и `j`этого массива. _Верните максимальное значение_ `(nums[i]-1)*(nums[j]-1)` .

**Пример 1:**

**Ввод:** nums = [3,4,5,2]
 **Вывод:** 12 
 **Пояснение:** Если выбрать индексы i=1 и j=2 (индексируются от 0), то получится максимальное значение, то есть (nums[1]-1)*(nums[2]-1) = (4-1)*(5-1) = 3*4 = 12. 

**Пример 2:**

**Ввод:** nums = [1,5,4,5]
 **Вывод:** 16
 **Пояснение:** Выбрав индексы i=1 и j=3 (индексируются от 0), вы получите максимальное значение (5-1)*(5-1) = 16.
