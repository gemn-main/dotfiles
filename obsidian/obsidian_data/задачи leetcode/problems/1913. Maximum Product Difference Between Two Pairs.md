```python
class Solution(object):
    def maxProductDifference(self, nums):
        nums.sort()
        return (nums[-1]*nums[-2]) - (nums[0]*nums[1])
```
Разница между двумя парами(a, b) определяется (c, d)как (a * b) - (c * d).

Например, разница между произведениями (5, 6)и (2, 7)составляет (5 * 6) - (2 * 7) = 16.
Дан целочисленный массив nums, выберите четыре различных индекса w, x, yи zтак, чтобы разность произведений между парами (nums[w], nums[x])и (nums[y], nums[z])была максимальной .

Верните максимальную разницу между этими произведениями .

 

Пример 1:

Вход: nums = [5,6,2,7,4]
 Выход: 34
 Пояснение: Мы можем выбрать индексы 1 и 3 для первой пары (6, 7) и индексы 2 и 4 для второй пары (2, 4). 
Разница произведений равна (6 * 7) - (2 * 4) = 34.
