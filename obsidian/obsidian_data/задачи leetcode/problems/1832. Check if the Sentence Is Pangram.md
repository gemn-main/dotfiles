```python
class Solution(object):
    def checkIfPangram(self, sentence):
        o = ''
        for i in range(len(sentence)):
            if sentence[i] not in o:
                o += sentence[i]
        return len(o) >= 26
```

Панграмма — это предложение, в котором каждая буква английского алфавита встречается хотя бы один раз.

Если задана строка, sentenceсодержащая только строчные буквы английского языка, вернуть значение, если это панграмма , или в противном случае. truesentencefalse

 

Пример 1:

Ввод: предложение = "thequickbrownfoxjumpsoverthelazydog"
 Вывод: true
 Пояснение: предложение содержит как минимум одну букву каждого английского алфавита.
