class MyClass:
    def __new__(cls, *args, **kwargs):
        print("Создание объекта")
        instance = super(MyClass, cls).__new__(cls)
        return instance

    def __init__(self, value):
        print("Инициализация объекта")
        self.value = value

Метод `__new__` вызывается **до создания объекта** и отвечает непосредственно за его создание. После этого созданный объект передается в метод `__init__`, который уже выполняет инициализацию (например, задает начальные значения атрибутов)
