выводит функцию 

если через объект класса надо вывести функцию  первым аргументом в функции надо передать *self*  
```python
class Point:  
    color = 'blue'  
    circle = 2  
  
  
    def set_coords(self):  
        print('hi skufindiai!') 
        
pt = Point()
pt.set_coords()

...
hi skufindiai!
```


в функции можно передавать дополнительные параметры:
```python
class Point:  
    color = 'blue'  
    circle = 2  
  
  
    def set_coords(self, x, y):  
        self.x = x  
        self.y = y  
        print(x, y)  
  
pt = Point()  
pt.set_coords(2, 1)

...
2 1
```
