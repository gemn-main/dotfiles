```python
class Point:  
    a = 21  
    color = 'blue'

prog = Point()
```

*prog* - Экземпляр класса он не имеет своих атрибутов, а лих хранит ссылки от Point

но если изменить атрибут в a то он присвоится ему:

```python
print(prog.__dict__)  
prog.color = 'red'  
print(prog.__dict__)

...
{}
{'color': 'red'}
```

еще можно добавить атрибут к Экземпляру класса:

```python
prog.lolik = 228
print(prog.__dict__)

...
{'color': 'red', 'lolik': 228}
```
