```cpp
int global_a = 5;

int main()
{
    int global_a = 10;
    std::cout << global_a << std::endl;
    return 0;
}
...
10
```

если добавить `::` в начале переменной  поиск будет глобальный:
```cpp
int global_a = 5;

int main()
{
    int global_a = 10;
    std::cout << ::global_a << std::endl;
    return 0;
}
...
5
```
