Очередь — это контейнер, работающий по принципу **FIFO (First In, First Out)** — первый пришёл, первый ушёл.
#### **1. Подключение библиотеки**
```cpp
#include <queue>
```
#### **2. Объявление очереди**
```cpp
queue<int> q;  // очередь целых чисел
```

#### **3. Основные операции**

| Метод     | Описание                        | Пример               |     |
| --------- | ------------------------------- | -------------------- | --- |
| `push()`  | Добавляет элемент в конец       | `q.push(10);`        |     |
| `pop()`   | Удаляет первый элемент          | `q.pop();`           |     |
| `front()` | Возвращает первый элемент       | `int x = q.front();` |     |
| `back()`  | Возвращает последний элемент    | `int y = q.back();`  |     |
| `empty()` | Проверяет, пуста ли очередь     | `if (q.empty()) ...` |     |
| `size()`  | Возвращает количество элементов | `int s = q.size();`  |     |
