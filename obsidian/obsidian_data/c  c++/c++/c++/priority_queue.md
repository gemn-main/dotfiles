# Очередь с приоритетом в C++ (priority_queue)

Очередь с приоритетом (priority queue) - это контейнерная структура данных, которая позволяет хранить элементы и обеспечивает быстрый доступ к элементу с наивысшим (или наинизшим) приоритетом.

```cpp
#include <queue>
#include <iostream>

int main() {
    // Создание очереди с приоритетом (по умолчанию - максимальный элемент наверху)
    std::priority_queue<int> pq;
    
    // Добавление элементов
    pq.push(30);
    pq.push(10);
    pq.push(20);
    pq.push(50);
    pq.push(40);
    
    // Извлечение элементов в порядке убывания
    while (!pq.empty()) {
        std::cout << pq.top() << " "; // Доступ к верхнему элементу
        pq.pop();                    // Удаление верхнего элемента
    }
    // Выведет: 50 40 30 20 10
}
```
