```cpp
ofstream ofs;
ifstream ifs;

ofs.open("ofs.dat");
ifs.open("ifs.dat");

cout << (ofs.is_open() ? "file is open" : "error open") << endl;
cout << (ifs.is_open() ? "file is open" : "error open") << endl;

...
file is open
error open
```

ofs - отрыт на запись поэтому он создан 
ifs - открыт на чтение поэтому он не создан 

можно убрать ...open(...):
```cpp
ofstream ofs("ofs.dat");
ifstream ifs("ifs.dat");
```
