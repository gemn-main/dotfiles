```cpp
inline bool sqr_root(double a, double b, double c, double &x1, double &x2)
{
    double D = b * b - 4 * a * c;
    if (D < 0)
        return false;

    x1 = (-b + sqrt(D)) / (2 * a);
    x2 = (-b - sqrt(D)) / (2 * a);
    return true;
}
int main()
{
    double x1{}, x2{};
    bool res_root = sqr_root(3, 10, 5, x1, x2);
    if (res_root)
    {
        cout << x1 << '\n'
             << x2 << endl;
    }
    return 0;
}
```
Ключевое слово **`inline`** в C++ используется для того, чтобы **попросить компилятор** подставить тело функции непосредственно в место её вызова, а не вызывать её как обычную функцию.


- **Ускорение работы программы**:  
    Для **малых функций** (например, геттеры, сеттеры, простые математические операции), где накладные расходы на вызов функции велики по сравнению с самой функцией. Вставка тела функции непосредственно в место вызова может сэкономить время.
    
- **Оптимизация производительности**:  
    Когда функция вызывается очень часто (например, внутри циклов), вызовы могут стать дорогими, и использование `inline` позволяет сэкономить время на каждом вызове.
    
- **Предотвращение множества определений функции**:  
    Если ты определяешь функцию внутри заголовочного файла, то компилятор может создать несколько определений функции (при включении заголовочного файла в разных исходных файлах). Использование `inline` сообщает компилятору, что ты хочешь, чтобы она была вставлена в каждом месте вызова, а не создавалась как отдельная функция.
