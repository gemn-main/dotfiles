### 1. Подключение библиотеки векторов

```cpp
#include <vector>
#include <iostream>
```

### 2. Создание вектора
```cpp
std::vector<int> vec; // Пустой вектор целых чисел
std::vector<int> vec2(10); // Вектор из 10 элементов, заполненных нулями
std::vector<int> vec3(5, 100); // Вектор из 5 элементов, каждый равен 100
std::vector<int> vec4 = {1, 2, 3, 4, 5}; // Вектор с инициализацией
```

### 3. Добавление и удаление элементов
```cpp
vec.push_back(10); // Добавляет 10 в конец
vec.push_back(20); 

vec.pop_back(); // Удаляет последний элемент

vec.insert(vec.begin(), 5); // Вставляет 5 в начало
vec.erase(vec.begin()); // Удаляет первый элемент
```


### 4. Доступ к элементам
```cpp
std::cout << vec[0];   // Безопасный, но не проверяет границы
std::cout << vec.at(0); // Проверяет границы (генерирует исключение при выходе за границы)
std::cout << vec.front(); // Первый элемент
std::cout << vec.back();  // Последний элемент
```


### 5. Размер и проверка вектора
```cpp
std::cout << vec.size();   // Текущий размер
std::cout << vec.capacity(); // Выделенная память
std::cout << vec.empty();  // true, если вектор пуст
```
