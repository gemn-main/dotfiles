```c
int main() {
	int lol = 3;
    _Bool res = lol > 0;
    printf("res = %d", res);
    return 0;
}
...
1
```
это тип делит все постигаемые значения на 2 группы:
0 - false
все остальные - true

в значение `_Bool` нельзя назначать true or false если не импортировать библиотеку `#include <stdbool.h>`   

```c
#include <stdbool.h>
int main() {
	bool res = false;
    printf("res = %d", res);
    return 0;
}
...
0
```



простая программа 
```c
#include <stdio.h>
#include <stdbool.h>
int main() {
    double y = -2.41;
    bool res = y >= -1 && y <= 4;
    printf("res = %d\n", res);
    return 0;
}
```

|| - или
