[[спецификаторы преобразований для scanf]]  
Эта функция считывает данные и вводит их в переменную 
### Синтаксис
```c
scanf("формат", &переменная1, &переменная2, ...);
```

- **"формат"** — это строка форматов, указывающая типы данных, которые нужно считать (например, `%d` для целых чисел, `%f` для чисел с плавающей точкой и т.д.).
- **&переменная** — это адрес переменной, в которую будут записаны введённые данные. **&** (амперсанд) означает, что передаётся адрес переменной, а не её значение.


```c
#include <stdio.h>
int main() {
    int val = 0;
    int res = scanf("%d", &val);
    printf("res = %d: val = %d", res, val);
    return 0;
}
...
151
res = 1: val = 151
```
в переменной res лежит количество успешно считанных переменных.


так как %d приводит значения к типу int то большие значения будут обрезаться чтобы этого избежать надо использовать [[модификаторы]]:
```c
#include <stdio.h>
int main() {
    long long val = 0;
    int res = scanf("%lld", &val);
    printf("res = %d: val = %lld", res, val);
    return 0;
}
...
12345678901234567
res = 1: val = 12345678901234567
```
