```c
enum colors
{
    red,
    green = 41,
    blue
};
int main()
{
    printf("red = %d, green = %d, blue = %d", red, green, blue);
    return 0;
}
red = 0, green = 41, blue = 42
```
Перечисление (enum) в языке C используется для создания набора именованных целочисленных констант, что упрощает код, делает его более читаемым и облегчает управление значениями, связанными с определенным набором состояний или опций.
