```python
int arr[10000000];

int main()
{
    printf("arr[12] = %d", arr[12]);

    return 0;
}
```
Глобальная переменная — это переменная, объявленная вне функций, доступная всем функциям в программе. Она сохраняет своё значение на протяжении всей работы программы и автоматически инициализируется нулем, если значение не указано.
