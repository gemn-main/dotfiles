```c
/*first program*/
#include <stdio.h>

int main(void)
{
    printf("Hello Wrold!\n"); //"Hello World"

}
```

программа всегда да начинается с функции main
*int* - означает что функция вернет целочисленное число 
*void* - если в функции нет параметров передается void
*return 0* - показывает что программа выполнила свою задачу и завершилась 
*\n* - перенос на новую строку
