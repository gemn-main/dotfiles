```c
int sum(int count, ...)
{
    int s = 0;
    va_list arg;
    va_start(arg, count);

    for(int i = 0; i < count; ++i){
        s += va_arg(arg, int);
    }
    va_end(arg);
    return s;
}


int main()
{   
    int res = sum(4,51,61,73,21);
    printf("res = %d",res);
    return 0;
}  
...
res = 206
```

`va_list` - указатель 
`va_start` - **адрес первого переменного аргумента**, который идет **в стеке сразу после именованного параметра `count`** 
`va_arg`  - **возвращает значение адреса arg и переносит указатель на следующую ячейку (то есть на 4 индекса так как это int)**
`va_end` - - Завершает работу с объектом `va_list` и освобождает ресурсы, если это необходимо.
