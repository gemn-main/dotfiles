```c
#include <stdio.h>
int main() {
    float a = 5.0;
    float b = 2.0;
    double res1 = 5.0 / 2.0;
    int c = 123;
    short d = 31;
    long res2 = d - c;
    printf("res1 = %f\n res2 = %d", res1, res2);
    return 0;
}
...
res1 = 2.500000
res2 = -92
```

при выполнении арифметических операций переменные приводятся к одному типу и при назначении их в переменную тоже


чтобы из 2 целочисленный переменных получить вещественное надо из преобразовать:
```c
#include <stdio.h>
int main() {
    int a = 12;
    int b = 7;
    double res = (double)a / (double)b;
    printf("res = %f", res);
    return 0;
}
...
res = 1.714286
```
