Объединение (**union**) в программировании используется для экономии памяти, когда переменная может хранить значения разных типов, но только одно из них в любой момент времени. Оно позволяет нескольким переменным делить одну и ту же область памяти, при этом тип данных определяется тем, какое поле используется.
```c
union tag_var
{
    char var_ch;
    int var_i;
    double var_d;
};

int main(void)
{
    union tag_var var;
    var.var_ch = 'k';
    var.var_i = 77;
    printf("var_ch = %c, var_i = %d, var_d = %f",
			     var.var_ch, var.var_i, var.var_d);
    return 0;
}

...
var_ch = M, var_i = 77, var_d = 0.000000
```

данные в нем храниться в 1 диапазоне, а это значит что верные значение будет только у последнего определенного параметра. в нашем случае этот параметр `var_i`
