```c
int main() {
    double a = -12.4, b = 1.51;
    double max_ab = a>b ? a:b;

    printf("%.3f", max_ab);
    return 0;
}
...
1.510
```
если условие ``a>b`` True в переменную вносится `a`, иначе - `b`    



пример:
```c
#include <math.h>
int main() {
    double a = -12.4, b = -31.51;

    double res_abs = a<b ? fabs(a):fabs(b);
    printf("%.3f", res_abs);
    return 0;
}
...
31.510
```
выводит самое меньшее число только в модуле 
`fads()` - модуль числа (библиотека - `math`)  


тернарная операция в тернарной операции:
```c
int main() {
    int a = 2, b = 14, c = -4;
    int max = (a>b) ? (a > c ? a : c) : (b>c)? b : c;
    printf("max = %d", max);
    return 0;
}
...
14
```
