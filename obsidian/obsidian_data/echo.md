просто выводит тест 

```bash
echo lol

```

```bash
echo *
-> anaconda3 bike code data Desktop Documents dotfiles Downloads gpu_fan_control.py grapejuice Music obsidian paru Pictures Public pytorch-rocm-docker rtl8188eu-dkms rtl88x2bu-dkms skrin steam Templates To topics venv Videos wallpaper yay
```

это произошло потому что символ `*` заместился на  последовательность любых символов в именах файлов в данной директории 


еще echo можно использовать как калькулятор 
```bash
echo $((2 + 4))
->6
```
