Метод `.get` в Python используется для получения значения из словаря по указанному ключу. Этот метод удобен тем, что позволяет избежать ошибок, возникающих при обращении к несуществующему ключу.

#### 1. Когда ключ существует
```python
data = {"name": "Alice", "age": 25}
print(data.get("name"))  # Вывод: Alice
```


#### 2. Когда ключ отсутствует
```python
data = {"name": "Alice", "age": 25} 
print(data.get("height"))  # Вывод: None
```
