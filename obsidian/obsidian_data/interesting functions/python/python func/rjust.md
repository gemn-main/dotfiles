Синтаксис:

```python
str.rjust(width, fillchar)
```
width — это общая длина строки, которую вы хотите получить (включая исходную строку и добавленные символы).
fillchar — символ, который будет добавлен в начало строки. По умолчанию это пробел.

```python
a = "IVAN"
newa = a.rjust(10, "-")
print(newa)
...
------IVAN
```
