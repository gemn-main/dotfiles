```python
class Solution(object):
    def minimumOperations(self, nums):
        max_num = max(nums)
        o = 0
        nums = list(set(nums))
        nums.sort()
        if nums[0] == 0:
            nums.pop(0)

        for i in range(len(nums)):
            if i > 0:
                k = nums[i] - nums[i-1]
            else:
                k = nums[i]
            max_num -= k
            o += 1
            print(max_num, nums[i])
            if max_num <= 0:
                return o
        return 0  
```
Вам дан неотрицательный целочисленный массив nums. За одну операцию вы должны:

Выберите положительное целое число x, которое xменьше или равно наименьшему ненулевому элементу в nums.
Вычтите xиз каждого положительного элемента в nums.
Верните минимальное количество операций, чтобы сделать каждый элементnums равным0 .

 

Пример 1:

Вход: nums = [1,5,0,3,5]
 Выход: 3
 Пояснение: 
В первой операции выберите x = 1. Теперь nums = [0,4,0,2,4]. 
Во второй операции выберите x = 2. Теперь nums = [0,2,0,0,2]. 
В третьей операции выберите x = 2. Теперь nums = [0,0,0,0,0].
Пример 2:

Ввод: nums = [0]
 Вывод: 0
 Пояснение: Каждый элемент в nums уже равен 0, поэтому никаких операций не требуется.
