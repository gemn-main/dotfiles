проверяет состоит ли строка только из цифр. Она возвращает True, если строка содержит только цифровые символы, и False в противном случае.

```python
# Пример 1: Содержит только цифры
s1 = "12345"
print(s1.isdigit())  # True

# Пример 2: Содержит не только цифры
s2 = "123a45"
print(s2.isdigit())  # False

# Пример 3: Пустая строка
s3 = ""
print(s3.isdigit())  # False

# Пример 4: Пробелы не считаются цифрами
s4 = "123 456"
print(s4.isdigit())  # False
```
