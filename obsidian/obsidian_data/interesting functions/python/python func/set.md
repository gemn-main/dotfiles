Это встроенный тип данных, который представляет собой **неупорядоченное множество уникальных элементов**. Он полезен для операций с уникальными элементами, удаления дубликатов и выполнения математических операций, таких как объединение, пересечение и разность.


## 2. Создание множества из последовательности
```python 
s = {1, 2, 3, 4}
print(s)  # Вывод: {1, 2, 3, 4}

s2 = set([1, 2, 2, 3])
print(s2)  # Вывод: {1, 2, 3}  (дубликаты автоматически удаляются)
```



## 3. Множество из строки
```python
s = set("hello")
print(s)  # Вывод: {'e', 'h', 'l', 'o'}
```
