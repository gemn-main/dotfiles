Функция `zip()` в Python используется для объединения нескольких итерируемых объектов (например, списков, кортежей, строк) в один, создавая из них последовательность кортежей. Каждая пара в кортежах формируется из элементов, имеющих одинаковый индекс в переданных итерируемых объектах.

```python
for num, char in zip(list1, list2):
    print(num, char)
# Вывод:
# 1 a
# 2 b
# 3 c
```
