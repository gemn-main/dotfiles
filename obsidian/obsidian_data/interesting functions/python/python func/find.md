Метод `find` в Python используется для поиска подстроки в строке. Этот метод возвращает индекс первого вхождения подстроки в строку. Если подстрока не найдена, возвращается `-1`.

---
### Различия между `find` и `index`

Если использовать метод `index`, то он вызывает ошибку (`ValueError`), если подстрока не найдена. Метод `find` более безопасен, если вы не уверены, что подстрока всегда присутствует.

---

```python
string.find(substring, start, end)
```
### Аргументы:

- **substring**: подстрока, которую нужно найти.
- **start** _(опционально)_: начальная позиция для поиска.
- **end** _(опционально)_: конечная позиция для поиска (не включительно).

### Возвращаемое значение:

- Индекс первого вхождения подстроки, если она найдена.
- `-1`, если подстрока отсутствует.

---

```python
text = "Hello, world!"
index = text.find("world")
print(index)  # Вывод: 7
```


```python
text = "Hello, world!"
index = text.find("Python")
print(index)  # Вывод: -1
```


```python
text = "Hello, world!"
index = text.find("o", 5, 10)
print(index)  # Вывод: 8
```
