```python
from collections import Counter

# Пример списка
arr = [1, 2, 2, 3, 3, 3, 4, 4, 5]

# Создаем объект Counter
counter = Counter(arr)

# Выводим результат
print(counter)

...
Counter({3: 3, 2: 2, 4: 2, 1: 1, 5: 1})

```
	
- В результате получается объект Counter, который является словарем, где ключи — это элементы списка, а значения — количество их повторений.
- В примере элемент 3 встречается 3 раза, элемент 2 — 2 раза, и так далее.





`counter.most_common(n)` — возвращает `n` наиболее часто встречающихся элементов и их количество:

```python
from collections import Counter

ar = [1,1,1,1,3,4,55,5,5,6]
arr = Counter(ar)
# Наиболее частый 1 элемен
print(arr.most_common(1))

...
[(1, 4)]
```
