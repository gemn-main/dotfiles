# Намеренья и цели 
1. установить цель чтения 
2. установить время чтения 
3. установить цель на прочтения (например прочитать книгу за месяц)
4. установить план в какое время читать
5. отмечать успехи 


# Активное чтение 
1. делать пометки
2. постанова вопросов
3. анализировать структуру и аргументацию автора 
4. обращать внимание на детали 

# Функции
 1. небольшие перерывы между темами (метод помидора)
 2. просмотреть текст перед прочтением (связать с уже прочитанным) **попутно делая пункт 4** 
 3. записывать только важное (своими словами)
 4. задавать вопросы книги во время чтений
 5. записывать то что не понимаешь, а потом гуглить
 6. награда за прочтение
