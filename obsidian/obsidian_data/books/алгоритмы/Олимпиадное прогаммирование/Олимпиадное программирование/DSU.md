**Disjoint Set Union**
структура данных непересекающихся множеств

DSU нужен для:
1. Объединения множеств (**union**)
2. Проверки, принадлежат ли элементы одному множеству (**find**)

## **Три ключевых компонента**
1. **`parent`** — массив "родителей" (каждый элемент указывает на своего предка)
2. **`find(x)`** — ищет "главного предка" (корень) элемента `x`
3. **`unite(x, y)`** — объединяет множества, содержащие `x` и `y`


в link - хранится следующий элемент пути или сам элемент если он представитель 
в size - для представителя хранится размер своего множества

## find (ищет представителя множества):
```cpp
int find(int x){
	while (x != link[x]) x = link[x];
	return x
}
```

## same (проверят принадлежат ли элементы одному множеству):
```cpp
bool same(int a, int b){
	return find(a) == find(b);
}
```
