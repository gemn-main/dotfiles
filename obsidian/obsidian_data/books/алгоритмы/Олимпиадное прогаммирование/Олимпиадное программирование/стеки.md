работает по правилам стека 
![[Pasted image 20250221155917.png]]
```cpp
int main()
{
    stack<int> s;
    s.push(41);              // [41]
    s.push(5);               // [41, 5]
    cout << s.top() << endl; // 5
    s.pop();                 // [41]
    cout << s.top() << endl; // 41
    return 0;
}
```
