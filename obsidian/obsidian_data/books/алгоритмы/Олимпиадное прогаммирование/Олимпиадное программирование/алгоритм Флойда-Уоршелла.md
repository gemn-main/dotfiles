### **Псевдокод**

```python
n = количество вершин
dist = матрица n x n, где dist[i][j] = вес ребра (i → j), или ∞, если ребра нет

for k от 0 до n-1:          # промежуточная вершина
    for i от 0 до n-1:       # начальная вершина
        for j от 0 до n-1:   # конечная вершина
            dist[i][j] = min(dist[i][j], dist[i][k] + dist[k][j])
```

1. **Инициализация:**
    
    - `dist[i][j]` = весу ребра `i → j`, если оно есть.
        
    - `dist[i][j] = ∞`, если ребра нет (и `i ≠ j`).
        
    - `dist[i][i] = 0` (расстояние от вершины к самой себе).
        
2. **Динамическое обновление:**  
    Для каждой вершины `k` (промежуточной) проверяем, улучшает ли путь `i → k → j` текущее расстояние `i → j`.
    
3. **Результат:**  
    После обработки всех `k`, `dist[i][j]` содержит кратчайший путь от `i` до `j`.
