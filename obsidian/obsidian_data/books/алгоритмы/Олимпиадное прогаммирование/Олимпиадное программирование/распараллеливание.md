процессоры могут выполнять несколько команд одновременно 
например:
```cpp
long long f = 1;
for (int i = 1; i <= n; i++){
	f = (f*i)%m;
}
```
выполняется в 2 раза дольше чем:

```cpp
long long f1 = 1;
long long f2 = 1;

for (int i = 1; i <= n; i += 2){
	f1 = (f1 * i) % m;
	f2 = (f2 * (i + 1)) % m;
}
long long  f = (f1 * f2) % m;
```
