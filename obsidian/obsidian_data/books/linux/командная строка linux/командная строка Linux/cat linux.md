Команда **`cat`** (от англ. *concatenate* — "объединять") в Linux используется для работы с содержимым файлов. Она выполняет три основные функции:

---

### **1. Вывод содержимого файла в терминал**
```bash
cat имя_файла
```
**Пример:**  
```bash
cat ~/document.txt  # показать текст файла document.txt
```
---
### **2. Объединение (конкатенация) файлов**
```bash
cat файл1 файл2 > новый_файл
```
**Пример:**  
```bash
cat part1.txt part2.txt > combined.txt  # объединяет part1 и part2 в combined.txt
```

**Важно:**  
- Если `новый_файл` уже существует, он будет **перезаписан**.  
- Чтобы **добавить** в конец файла, используйте `>>`:  
  ```bash
  cat part3.txt >> combined.txt  # дописывает part3 в конец combined.txt
  ```

---

### **3. Создание файлов и ввод текста**
```bash
cat > новый_файл
```
После ввода команды можно писать текст. Для сохранения нажмите:  
1. `Ctrl + D` (завершение ввода).  
2. `Ctrl + C` (отмена).  

**Пример:**  
```bash
cat > greeting.txt  
Привет, мир!  # Вводим текст  
Ctrl+D        # Сохраняем
```

---

### **Дополнительные опции**
| Опция | Описание |
|-------|----------|
| `-n`  | Нумеровать строки при выводе: `cat -n file.txt` |
| `-b`  | Нумеровать только непустые строки: `cat -b file.txt` |
| `-s`  | Сжимать несколько пустых строк в одну: `cat -s file.txt` |
| `-A`  | Показать непечатаемые символы (табы, переносы): `cat -A file.txt` |

**Пример с опциями:**  
```bash
cat -n -b ~/script.py  # вывести файл с нумерацией строк
```

---

### **Комбинации с другими командами**
1. **Постраничный просмотр** (через `less`):  
   ```bash
   cat large_file.log | less
   ```
2. **Поиск текста** (через `grep`):  
   ```bash
   cat server.log | grep "ERROR"
   ```
3. **Подсчёт строк** (через `wc`):  
   ```bash
   cat file.txt | wc -l
   ```
